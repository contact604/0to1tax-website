# ⚡ DNS 오류 1분 요약

## 🚨 오류
```
DNS_PROBE_FINISHED_NXDOMAIN
0to1tax.com - 사이트에 연결할 수 없음
```

---

## 🎯 원인
**도메인(0to1tax.com)과 웹사이트가 연결 안 됨!**

---

## ✅ 해결 (3가지 중 선택)

### 1. 배포 URL 사용 ⭐ (즉시!)
```
Publish 탭 → URL 확인 → 그 주소 사용
예: https://iadkczem.gensparkspace.com

✅ 장점: 즉시 작동, 설정 불필요
❌ 단점: 주소가 길다
```

### 2. Cloudflare 설정 ⭐⭐⭐ (2~24시간)
```
1. cloudflare.com 가입
2. 도메인 추가 (0to1tax.com)
3. DNS 레코드 추가
4. 네임서버 변경
5. 대기

✅ 장점: 무료 SSL, 빠른 속도, 보안
❌ 단점: 설정 복잡, 대기 필요

📖 가이드: CLOUDFLARE_SETUP_GUIDE.md
```

### 3. Squarespace DNS ⭐⭐ (2시간)
```
1. Squarespace 로그인
2. DNS 설정
3. CNAME/A 레코드 추가
4. 대기

✅ 장점: 비교적 간단
❌ 단점: SSL 문제 가능

📖 가이드: 도메인_연결_간단가이드.md
```

---
