## 📱 **모바일 Solution 개선 상세**

### **Before (문제)**
```
┌─────────────────────┐
│  Problem Box        │
│  투자사의 재무실...  │ ← 글씨 반만 보임
│  (잘림)             │
├─────────────────────┤
│  Solution Box       │
│  투자유치에 유리... │ ← 글씨 반만 보임
│  (잘림)             │
└─────────────────────┘
```

### **After (해결)** ✅
```
┌─────────────────────┐
│  Problem Box        │
│  투자사의 재무실사가│
│  두려워요           │ ← 전체 보임!
├─────────────────────┤
│  Solution Box       │
│  (스크롤 가능 ↓)    │
│  투자유치에 유리한  │
│  재무제표를 작성해요│ ← 전체 보임!
└─────────────────────┘
    👆 스와이프로 이동
```

## 🎯 **스와이프 제스처 동작**

### **사용자 경험**
```
← 오른쪽 스와이프 = 이전 슬라이드
→ 왼쪽 스와이프   = 다음 슬라이드
```

### **기술 구현**
- `touchstart`: 터치 시작 좌표 저장
- `touchend`: 터치 종료 좌표 저장
- 수평 이동 > 수직 이동 시에만 작동
- 최소 이동 거리: 50px
- 스크롤과 충돌하지 않음
