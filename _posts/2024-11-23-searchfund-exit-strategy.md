## 👋 Intro: 왜 지금 '서치펀드'인가?

최근 40–60대 중소기업 창업자들로부터 "은퇴 시 회사는 어떻게 넘기는 게 좋을까요?"라는 질문을 정말 많이 받습니다.
한국은 이제 본격적으로 오너 1세대의 대규모 은퇴 시대로 들어섰고, "후계자 없음 → 매각"이 보편적 흐름이 되고 있습니다.

문제는…

<ul>
<li>자녀 세대가 승계를 원하지 않아서</li>
<li>회사를 맡길 적임자가 없어서</li>
<li>매각하려니 PEF는 너무 크고</li>
<li>중개 플랫폼은 너무 표준화되어 있고</li>
<li>기업브로커는 검증이 어려워서</li>
</ul>

창업자 입장에서 "마음 편하게 넘길 곳"이 없다는 점이죠.

이때 등장한 새로운 대안이 바로 서치펀드(Search Fund) 입니다.

서치펀드는 '0에서 1'을 이미 만든 중소기업 창업자들이 은퇴와 승계를 준비할 때 선택할 수 있는 가장 현실적인 Exit 전략입니다.

---

## 1. 서치펀드는 무엇인가?

서치펀드는 미국에서 시작된 중소기업 인수·승계 모델입니다.
젊은 MBA 출신(주로 30대 초반)이 투자자 자금을 모아 회사를 인수하고, 본인이 직접 CEO로 들어가는 구조입니다.

### ✔ 서치펀드 4단계 구조

| 단계 | 설명 |
|------|------|
| ① 탐색(Search) | 투자자에게 탐색자금(2년치 운영비) 조달 |
| ② 인수(Acquisition) | 인수 대상 기업 발굴 → 매수단가 협상 → 인수 |
| ③ 경영(Operation) | 서치펀드 창업자가 CEO로 직접 경영 |
| ④ Exit | 5–7년 운영 후 매각/재자본화로 회수 |

이 구조 덕분에
"자기 회사를 맡기고 떠나는 창업자"와
"CEO가 되고 싶은 젊은 경영자"가
서로 win–win 할 수 있는 매우 특별한 모델로 자리 잡았습니다.

## 2. 서치펀드는 어떤 기업을 인수할까? (해외 데이터 기반)

IESE·Stanford Search Fund Report 기준 실제 인수 기업 데이터:

### ✔ 인수 기업 평균 스펙

| 항목 | 수치 |
|------|------|
| 매출 | 80억~300억 원 |
| EBITDA | 10억~50억 원 |
| EBITDA 마진 | 15~30% |
| 직원 수 | 20~100명 |
| 오너 1세대 기업 | 90% 이상 |
| 산업 | B2B 서비스, 제조, 유통, 헬스케어 |

### ✔ 핵심 조건

<ul>
<li>꾸준히 현금을 벌고</li>
<li>변동성이 적고</li>
<li>오너가 고령이고</li>
<li>승계가 어렵고</li>
<li>운영이 복잡하지 않은 회사</li>
</ul>

→ 즉, 한국에서 우리가 흔히 보는 '잘 되는 중소기업'이 서치펀드의 핵심 타겟층입니다.

## 4. 한국에서 서치펀드가 필요한 이유

한국은 세계적으로 가장 빠르게 은퇴하는 창업자 구조를 갖고 있습니다.

### 📌 한국 상황 요약

<ul>
<li>중소기업의 30%가 오너 60대 이상</li>
<li>자녀 승계율 30% 이하</li>
<li>매각이 어려운 기업이 전체의 70%</li>
<li>"사업은 괜찮은데 승계를 못해서 문 닫는" 기업 증가</li>
</ul>

즉, 회사를 계속 운영할 사람만 찾으면 살아남을 기업이 정말 많다는 뜻입니다.

### ✔ 서치펀드는 이 문제를 정확히 해결합니다

→ "젊고 검증된 경영자(30대)에게 회사 전체를 넘기고 은퇴할 수 있는 구조"

PEF처럼 지배구조가 강압적이지 않고,
브로커처럼 단발성 매각도 아닙니다.

승계 + 경영 + 성장
이 3가지를 동시에 해결해주는 모델입니다.

## 5. 서치펀드 vs PEF(사모펀드) 간단 비교

| 구분 | 서치펀드 | PEF(사모펀드) |
|------|----------|--------------|
| CEO | 젊은 운영자가 직접 CEO | 기존 경영진 유지하거나 파견 |
| 인수 규모 | 20억~80억(한국 기준) | 200억~수천억 |
| 방식 | 경영승계 중심 | 재무적 투자/구조조정 중심 |
| 창업자 입장 | 철학·문화 승계 용이 | Exit은 가능하나 문화 단절 |
| 투자자 | 엔젤·HNW·펀드 혼합 | 기관 중심 |
| 적합 기업 | 20~200억 매출 | 300억~수천억 매출 |

즉, 서치펀드는 '작지만 탄탄한 기업'을 위한 모델,
PEF는 '규모 있는 기업'을 위한 모델입니다.

## 6. 한국형 서치펀드의 미래

한국에서도 다음 분야가 서치펀드 핵심 대상이 될 것으로 보입니다:

<ul>
<li>B2B 기술 서비스</li>
<li>제조업(정밀, 부품, 기계)</li>
<li>유통·물류</li>
<li>헬스케어 서비스</li>
<li>브랜드 파워 있는 중소기업</li>
</ul>

특히 "후계자 부재 + 현금창출력 견고 + 고령 창업자" 조합은
서치펀드에게 완벽한 환경입니다.

<summary>▶ Q1. 한국에도 서치펀드가 있나요?</summary>

네. 최근 2~3년간 한국에서도 도입 시도가 증가하고 있으며, 중소기업 매각·승계 시장에서 점점 관심이 커지고 있습니다.

<details>
<summary>▶ Q2. 어떤 규모의 회사가 적합한가요?</summary>

매출 20억~200억 사이, EBITDA 5억 이상이면 매우 적합합니다.

</details>

<details>
<summary>▶ Q3. 창업자가 서치펀드에 매각하면 장점은?</summary>

<ul>
<li>신뢰 가능한 젊은 CEO에게 승계</li>
<li>기업 문화가 유지됨</li>
<li>오너의 철학이 이어짐</li>
<li>단계적 Exit 가능</li>
</ul>

</details>

## 8. 마무리: 서치펀드는 은퇴 창업자의 새로운 선택지

서치펀드는 단순한 매각 방식이 아니라,
"창업자가 평생 만든 회사를 잘 이어줄 사람을 찾는 승계 모델"입니다.

한국에서도 앞으로 가장 빠르게 성장할 M&A 형식 중 하나입니다.
