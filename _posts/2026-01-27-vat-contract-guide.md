# 부가가치세 리스크를 줄이는 계약서 작성법
선금·중도금·잔금 조건 명시 방법

부가가치세 공급시기는 계약서 작성 방법에 따라 달라집니다. 같은 거래라도 계약서를 어떻게 쓰느냐에 따라 세금계산서 발행 시기와 납부 시기가 변경됩니다.

## 부가가치세법에서 인정하는 계약서 조건

부가가치세법 시행령 제53조 제1항 제3호는 계약서에 대금 청구 시기와 지급 시기가 구분되어 있고 그 간격이 30일 이내인 경우, 세금계산서 발행일을 공급시기로 인정합니다.

이 특례를 활용하려면 계약서에 세 가지 요건을 충족해야 합니다.

## 계약서 작성 필수 요건

| 요건 | 내용 | 예시 |
|------|------|------|
| **청구·지급 구분** | 청구 시기와 지급 시기를 명확히 구분 기재 | "계약일 청구, 청구일로부터 15일 이내 지급" |
| **30일 이내** | 청구일부터 지급일까지 기간 30일 이내 | 15일, 30일 가능 / 45일, 60일 불가 |
| **세금계산서 명시** | 세금계산서 발행 시기를 계약서에 기재 | "각 대금 청구 시 세금계산서 발행" |

### 잘못된 계약서 vs 올바른 계약서

**잘못된 예시:**
```
제5조 (대금 지급)
- 선금: 계약금액의 30%
- 중도금: 계약금액의 30%
- 잔금: 계약금액의 40%
```

청구 시기와 지급 시기가 구분되지 않아 특례 적용 불가합니다.

**올바른 예시:**
```
제5조 (대금 지급)
- 선금 30%: 계약일 청구, 청구일로부터 15일 이내 지급
- 중도금 30%: 50% 완료 시 청구, 청구일로부터 15일 이내 지급
- 잔금 40%: 완료 시 청구, 청구일로부터 15일 이내 지급

제6조 (세금계산서)
갑은 각 대금 청구 시 세금계산서를 발행한다.
```

## 특례 적용 여부에 따른 공급시기

| 계약서 작성 방법 | 공급시기 | 특례 적용 |
|----------------|---------|----------|
| 청구·지급 구분 (30일 이내) | 세금계산서 발행일 | O |
| 청구·지급 구분 (30일 초과) | 재화·용역 공급일 | X |
| 단순 비율만 기재 | 공급일 또는 입금일 | X |

### 실무 사례

**계약 조건:**
- 선금 30%: 계약일 청구, 15일 이내 지급
- 중도금 30%: 50% 완료 시 청구, 15일 이내 지급
- 잔금 40%: 완료 시 청구, 15일 이내 지급

**실제 진행:**
- 2026.01.10: 계약 체결 → 선금 세금계산서 발행
- 2026.01.25: 선금 입금
- 2026.02.15: 50% 완료 → 중도금 세금계산서 발행
- 2026.03.01: 중도금 입금
- 2026.04.01: 완료 → 잔금 세금계산서 발행
- 2026.04.15: 잔금 입금

**공급시기:** 각 세금계산서 발행일 (입금일과 무관)
- 선금: 2026.01.10
- 중도금: 2026.02.15
- 잔금: 2026.04.01

## 계약서에 추가하면 좋은 조항

### 세금계산서 발행 조항
```
제○조 (세금계산서)
갑은 각 대금 청구 시 전자세금계산서를 발행하며,
세금계산서는 청구일 기준으로 발행한다.
```

### 지연이자 조항
```
제○조 (지연이자)
을이 지급 기한을 초과한 경우 연 ○%의 지연이자를 부담한다.
```

### 공급시기 명시 조항
```
제○조 (부가가치세)
본 계약의 부가가치세 공급시기는 각 대금 청구일로 하며,
을의 대금 미지급이 있더라도 갑의 부가가치세 신고 의무는
청구일 기준으로 발생한다.
```

## 자주 하는 실수

**실수 1: 청구와 지급 혼동**

"선금 30%를 계약 시 지급한다" → 청구 시기 누락

올바른 표현: "계약일에 청구하며, 청구일로부터 15일 이내 지급한다"

**실수 2: 30일 초과 설정**

"청구일로부터 45일 이내 지급" → 특례 적용 불가

**실수 3: 일부 대금만 조건 명시**

선금만 조건을 명시하고 중도금·잔금은 조건 없음 → 전체 특례 적용 불가

**실수 4: 공급 완료와 청구 시점 혼동**

"완료 시 잔금 지급" → 청구 시기 불명확

올바른 표현: "완료 시 청구하며, 청구일로부터 15일 이내 지급한다"

## 계약서 검토 체크리스트

**필수 확인:**
- [ ] 청구 시기가 명확한가
- [ ] 지급 시기가 명확한가
- [ ] 청구~지급 기간이 30일 이내인가
- [ ] 세금계산서 발행 시기가 명시되어 있는가

**권장 확인:**
- [ ] 지연이자 조항이 있는가
- [ ] 공급시기 명시 조항이 있는가

## 0to1tax 정리

계약서는 거래 조건만 정하는 문서가 아닙니다. 부가가치세 공급시기와 세무 리스크를 결정하는 법적 근거입니다.

선금·중도금·잔금 계약에서 청구 시기와 지급 시기를 명확히 구분하고 그 기간을 30일 이내로 설정하면, 세금계산서 발행일을 공급시기로 인정받습니다. 이는 현금 흐름과 세금 납부 시기를 분리할 수 있다는 의미입니다.

실무에서 가장 흔한 실수는 단순히 "선금 30%, 중도금 30%, 잔금 40%"만 적는 것입니다. 반드시 청구 시기와 지급 시기를 모두 명시해야 특례를 적용받을 수 있습니다.
