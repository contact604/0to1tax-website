연결재무제표를 작성할 때 그룹 내부에서 발생한 거래는 모두 제거해야 합니다. A사가 B사에 상품을 판매했다면, 그룹 관점에서는 외부 매출이 아니라 내부 이동일 뿐이기 때문입니다. 오늘은 내부거래 제거의 원리와 실무 처리 방법을 자세히 알려드리겠습니다.

## 왜 내부거래를 제거해야 하나?

연결재무제표는 그룹 전체를 하나의 경제적 실체로 표시하는 것입니다. 따라서 그룹 내부에서 발생한 거래는 실질적으로 자기 자신과의 거래이므로 제거해야 합니다.

### 실무 사례

A사가 B사에 제품을 50억 원에 판매했습니다.

**A사 입장 (판매자)**:
- 매출 50억 원 증가
- 매출원가 30억 원 증가
- 매출총이익 20억 원

**B사 입장 (구매자)**:
- 매입 50억 원 (매출원가에 포함)

만약 이 거래를 제거하지 않으면 어떻게 될까요?

**조정 전 연결손익계산서**:
```
매출: A사 300억 + B사 150억 = 450억
매출원가: A사 180억 + B사 90억 = 270억
```

하지만 그룹 관점에서 A사가 B사에 판매한 50억 원은 외부 매출이 아닙니다. 따라서 이를 제거해야 합니다.

## 내부매출·매입 제거

### 기본 원리

그룹 내 한 회사의 매출은 다른 회사의 매입(매출원가)이 됩니다. 따라서 매출과 매출원가를 동시에 제거합니다.

### 연결 조정 분개

```
차) 매출 (A사)           50억
  대) 매출원가 (B사)      50억
```

이 분개를 통해:
- A사의 매출 50억 원이 감소
- B사의 매출원가 50억 원이 감소
- 그룹 전체 매출과 매출원가가 각각 50억 원씩 감소

### 조정 후 연결손익계산서

```
매출: 450억 - 50억 = 400억 (외부 매출만)
매출원가: 270억 - 50억 = 220억
매출총이익: 180억 (변동 없음)
```

**중요**: 매출총이익은 변동이 없습니다. 내부거래 제거는 매출과 매출원가를 동시에 줄이므로, 이익에는 영향을 주지 않습니다.

## 복잡한 내부거래

실무에서는 여러 회사 간 다양한 내부거래가 발생합니다.

### 다단계 내부거래

```
A사 → B사: 제품 판매 50억
B사 → C사: 가공 후 재판매 80억
C사 → 외부: 최종 판매 120억
```

**연결 조정**:

1. A사 → B사 거래 제거:
```
차) 매출 (A사)           50억
  대) 매출원가 (B사)      50억
```

2. B사 → C사 거래 제거:
```
차) 매출 (B사)           80억
  대) 매출원가 (C사)      80억
```

최종적으로 연결재무제표에는 C사가 외부에 판매한 120억 원만 매출로 남습니다.

## 내부 용역거래

제품 판매뿐만 아니라 용역 제공도 내부거래에 해당합니다.

### 사례: 경영 자문료

A사가 B사에 경영 자문 서비스를 제공하고 10억 원을 청구했습니다.

**A사 (용역 제공자)**:
- 용역수익 10억 원

**B사 (용역 수취자)**:
- 판관비 (자문료) 10억 원

### 연결 조정 분개

```
차) 용역수익 (A사)       10억
  대) 판관비 (B사)        10억
```

그룹 관점에서는 내부 용역 제공이므로, 수익과 비용을 모두 제거합니다.

## 내부 이자수익·비용

그룹 내 대여거래로 발생한 이자도 제거해야 합니다.

### 사례: 내부 대여금

A사가 B사에 20억 원을 대여하고, 연 5% 이자를 받습니다. 1년간 이자 1억 원이 발생했습니다.

**A사 (대여자)**:
- 이자수익 1억 원

**B사 (차입자)**:
- 이자비용 1억 원

### 연결 조정 분개

```
차) 이자수익 (A사)       1억
  대) 이자비용 (B사)      1억
```

그룹 관점에서는 내부 자금 이동이므로, 이자수익과 이자비용을 모두 제거합니다.

## 내부 배당금

종속회사가 지배회사에 배당을 지급한 경우도 제거합니다.

### 사례: 내부 배당

B사가 당기순이익 20억 원 중 10억 원을 A사에 배당으로 지급했습니다.

**A사 (수령자)**:
- 배당수익 10억 원

**B사 (지급자)**:
- 이익잉여금 감소 10억 원

### 연결 조정 분개

```
차) 배당수익 (A사)       10억
  대) 배당금 (B사)        10억
```

그룹 관점에서 종속회사의 배당은 내부 자금 이동이므로 제거합니다. 연결재무제표에는 그룹 외부 주주에게 지급한 배당만 표시됩니다.

## 내부거래 집계 시스템

실무에서 내부거래를 정확히 파악하고 제거하려면 체계적인 집계 시스템이 필요합니다.

### 내부거래 집계표 (예시)

| 판매자 | 구매자 | 거래 유형 | 금액 | 제거 여부 |
|--------|--------|-----------|------|-----------|
| A사 | B사 | 제품 판매 | 50억 | ✅ |
| A사 | C사 | 용역 제공 | 10억 | ✅ |
| B사 | C사 | 제품 판매 | 30억 | ✅ |
| A사 | B사 | 이자수익 | 1억 | ✅ |
| B사 | A사 | 배당금 | 10억 | ✅ |

매 분기 또는 매 연도 종료 시점에 각 회사로부터 내부거래 내역을 수집하여 이런 집계표를 작성합니다. 엑셀이나 ERP 시스템을 활용하면 효율적입니다.

### 실무 팁: 거래처 코드 관리

그룹 내 각 회사에 거래처 코드를 부여하여 회계 시스템에서 자동으로 내부거래를 식별할 수 있도록 합니다.

예시:
- A사: IC-001
- B사: IC-002
- C사: IC-003

거래처 코드가 "IC-"로 시작하면 내부거래로 자동 분류하여 연결 조정 시 누락을 방지할 수 있습니다.

## 제거하지 않아도 되는 거래

모든 내부거래를 제거하는 것은 아닙니다. 다음 경우는 제거하지 않습니다.

### 1. 외부와의 거래

A사와 B사가 모두 같은 외부 업체에서 상품을 구매하는 경우, 이는 내부거래가 아니므로 제거하지 않습니다.

### 2. 정상적인 시장 가격으로 거래 후 외부 판매 완료

A사가 B사에 제품을 판매하고, B사가 외부에 판매를 완료했다면, 미실현손익이 없으므로 추가 조정이 불필요합니다. (단, 매출·매입은 여전히 제거)

### 3. 연결 범위 밖 거래

연결 범위에 포함되지 않은 회사와의 거래는 내부거래가 아니므로 제거하지 않습니다.

예시:
- A사가 D사(지분법 적용 회사)와 거래 → 제거하지 않음
- A사가 E사(단순 투자 회사)와 거래 → 제거하지 않음

## 주의사항

### 1. 거래 시점 차이

A사가 12월 30일에 B사에 판매했지만, B사가 1월 5일에 매입으로 기록한 경우, 거래 시점 차이가 발생합니다. 이 경우 기간을 일치시켜 조정해야 합니다.

### 2. 금액 불일치

A사의 매출 50억 원과 B사의 매입 51억 원이 서로 다른 경우, 원인을 파악하여 조정해야 합니다. 운송비, 보험료 등이 포함되었을 수 있습니다.

### 3. 외화 거래

내부거래가 외화로 이루어진 경우, 환율 차이로 인해 금액이 다를 수 있습니다. 동일한 환율을 적용하여 조정합니다.

## 결론

내부거래 제거는 연결재무제표 작성의 핵심입니다. 그룹 내부에서 발생한 매출·매입, 용역, 이자, 배당 등 모든 거래를 제거하여 그룹 전체의 순수한 외부 거래만 표시해야 합니다. 실무에서는 내부거래 집계 시스템을 구축하고, 거래처 코드를 활용하여 누락 없이 정확히 제거하는 것이 중요합니다.

다만 내부거래를 제거해도 매출총이익이나 당기순이익에는 영향이 없습니다. 수익과 비용을 동시에 제거하기 때문입니다. 연결재무제표는 그룹의 외부 거래만 표시하여 투자자와 채권자에게 정확한 정보를 제공하는 것이 목적입니다.
