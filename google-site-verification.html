구글 서치 콘솔 소유 확인 페이지

이 페이지는 구글 서치 콘솔 사이트 소유 확인을 위한 페이지입니다.

🔍 구글 검색 등록 방법

1단계: 구글 서치 콘솔 접속

👉 https://search.google.com/search-console

2단계: 속성 추가

  1. 구글 계정으로 로그인
  2. "속성 추가" 버튼 클릭
  3. "URL 접두어" 선택 (추천)
  4. 사이트 URL 입력: https://0to1tax.com

3단계: 사이트 소유 확인 (여러 방법 중 선택)

방법 1: HTML 태그 (가장 쉬운 방법 - 추천)

  1. 구글에서 제공하는 메타 태그 복사
  2. index.html 파일의 <head> 태그 안에 붙여넣기
  3. 예시: <meta name="google-site-verification" content="코드" />
  4. 파일 업로드 후 구글에서 확인 버튼 클릭

방법 2: HTML 파일 업로드

  1. 구글에서 제공하는 HTML 파일 다운로드
  2. 파일명: googleXXXXXXXXXXXX.html
  3. 웹사이트 루트 디렉토리에 업로드
  4. 구글에서 확인 버튼 클릭

방법 3: Google Analytics

Google Analytics가 이미 설치되어 있다면 자동으로 확인 가능

방법 4: Google Tag Manager

Google Tag Manager가 이미 설치되어 있다면 자동으로 확인 가능

💡 TIP: 이 파일의 <head> 태그에 있는 메타 태그를 구글에서 제공한 실제 인증 코드로 변경하세요.

4단계: 사이트맵 제출

  1. 사이트 소유 확인 완료 후
  2. 왼쪽 메뉴에서 "Sitemaps" 선택
  3. 새 사이트맵 추가 입력란에 입력: sitemap.xml
  4. 제출 버튼 클릭

📌 전체 URL: https://0to1tax.com/sitemap.xml

5단계: URL 검사 (선택사항 - 빠른 색인을 원할 때)

  1. 상단의 URL 검사 도구 사용
  2. 메인 페이지 URL 입력: https://0to1tax.com
  3. "색인 생성 요청" 클릭

⚡ 일반적으로 몇 시간~며칠 내에 색인이 생성됩니다.

📊 중요한 메뉴들

  • 실적 (Performance): 검색 트래픽 분석
  • URL 검사: 특정 페이지 색인 상태 확인
  • 커버리지 (Coverage): 색인 생성된 페이지 확인
  • 개선사항 (Enhancements): 모바일 사용성, Core Web Vitals 등

✅ 검색 노출 확인

등록 후 며칠~2주 정도 소요됩니다.

구글 검색창에 다음과 같이 검색해보세요:

  • site:0to1tax.com - 사이트 전체 색인 확인
  • Zero to one TAX - 실제 검색어로 확인
  • "Zero to one TAX" 스타트업 세무 - 정확한 키워드 확인

⚠️ 주의사항

  • 소유 확인 메타 태그는 삭제하지 마세요 (계속 필요함)
  • robots.txt가 크롤링을 차단하지 않는지 확인하세요
  • 사이트맵은 주기적으로 업데이트하세요
  • 새 페이지 추가 시 URL 검사 도구로 빠른 색인 요청 가능
