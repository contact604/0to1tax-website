# Images Folder

## CEO Profile Image

이범기 회계사님의 프로필 사진을 `ceo-profile.jpg` 파일명으로 이 폴더에 추가해주세요.

### 이미지 요구사항:
- 파일명: `ceo-profile.jpg` 또는 `ceo-profile.png`
- 권장 크기: 400x400px 이상 (정사각형)
- 형식: JPG, PNG

### 이미지 추가 방법:
1. 이미지를 웹 호스팅 서비스에 업로드 (예: imgur.com, Google Drive 등)
2. 공개 URL을 받아서 사용
3. 또는 base64 인코딩하여 직접 HTML에 삽입

현재 설정: 이미지가 없으면 자동으로 아이콘이 표시됩니다.
