# ⚡ 검색 노출 빠른 시작 가이드 (5분 완성)

> **최소 시간으로 검색 엔진 등록하기**  
> **소요 시간**: 5분  
> **난이도**: ⭐☆☆☆☆

---

## 🎯 핵심 3단계

### 1️⃣ 구글 서치 콘솔 등록 (2분)

1. 👉 **[Google Search Console](https://search.google.com/search-console)** 접속
2. "속성 추가" → "URL 접두어" → `https://0to1tax.com` 입력
3. 소유권 확인 방법 선택:
   - **HTML 태그** (추천): 메타 태그를 `index.html`의 `<head>` 안에 추가
   - **HTML 파일**: 제공된 파일을 루트에 업로드
4. 확인 완료 후 "Sitemaps" → `sitemap.xml` 제출

✅ **완료!**

---

### 2️⃣ 네이버 서치어드바이저 등록 (2분)

1. 👉 **[네이버 서치어드바이저](https://searchadvisor.naver.com/)** 접속
2. "웹마스터 도구" → "사이트 추가" → `https://0to1tax.com` 입력
3. 소유권 확인 방법 선택:
   - **HTML 태그** (추천): 메타 태그를 `index.html`의 `<head>` 안에 추가
   - **HTML 파일**: 제공된 파일을 루트에 업로드
4. 확인 완료 후 "요청" → "사이트맵 제출" → `https://0to1tax.com/sitemap.xml`

✅ **완료!**

---

### 3️⃣ 검색 노출 확인 (1분)

#### 즉시 확인 (색인 요청)
- **구글**: URL 검사 도구에서 "색인 생성 요청"
- **네이버**: "요청" → "웹페이지 수집 요청"

#### 실제 검색 (며칠~2주 후)
```
구글: site:0to1tax.com
네이버: site:0to1tax.com
```

---

## 📝 index.html에 추가할 코드

구글과 네이버 인증 후 받은 메타 태그를 다음 위치에 추가:

```html
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    
    <!-- Primary Meta Tags -->
    <meta name="robots" content="index, follow">
    
    <!-- 🔵 여기에 구글 인증 메타 태그 추가 -->
    <meta name="google-site-verification" content="구글에서_받은_코드" />
    
    <!-- 🟢 여기에 네이버 인증 메타 태그 추가 -->
    <meta name="naver-site-verification" content="네이버에서_받은_코드" />
    
    <!-- 나머지 메타 태그들... -->
</head>
```

## ✅ 이미 준비된 것들

다음 파일들이 이미 최적화되어 있습니다:

- ✅ **robots.txt** - 검색봇 크롤링 허용 설정

- ✅ **메타 태그** - SEO 최적화 완료
- ✅ **구조화 데이터** - 사업장 정보 포함
