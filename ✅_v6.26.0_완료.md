## 💡 구조 통일의 장점

### 1. 일관성 ⬆️⬆️⬆️
```
그로스 플랜:
기본 (18만원) + 추가 (별도 문의)

스케일업 플랜:
기본 (18만원) + 추가 (별도 문의)

→ 동일한 구조로 이해 쉬움 ✅
```

### 2. 비교 용이 ⬆️⬆️
```
그로스 vs 스케일업:
├─ 공통: 엔터프라이즈 플랜 (18만원)
├─ 차이: 추가 서비스만 다름
│  ├─ 그로스: 스톡옵션, IR DECK, 라운드 유치...
│  └─ 스케일업: PA, 기업가치평가, 감사...
└─ 결과: 선택이 명확해짐 ✅
```

### 3. 가격 투명성 ⬆️⬆️⬆️
```
BEFORE: "문의 필수"
사용자: "얼마인지 모르겠네..." 😕

AFTER:  "18만원 + 별도 문의"
사용자: "기본은 18만원이구나" 😊
        "추가는 따로 문의하면 되고"
```
