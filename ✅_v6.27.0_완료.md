## 💡 사용자 경험 개선

### 1. 메뉴 명확성 ⬆️⬆️⬆️
```
BEFORE:
- "Launch Plans" - 영어라 이해 어려움
- "Recommended Plans" - 메뉴에 없음

AFTER:
- "성장여정" - 한글로 명확
- "런치 플랜" - 한글로 접근성 향상
```

### 2. 중복 제거 ⬆️⬆️
```
BEFORE:
- Launch Plans 섹션 (스타터/프로페셔널/엔터프라이즈)
- Recommended Plans 섹션 (스타터/그로스/스케일업)
→ 혼란스러움 ❌

AFTER:
- 런치 플랜 섹션만 (스타터/그로스/스케일업)
→ 명확함 ✅
```

### 3. 접근성 향상 ⬆️⬆️
```
BEFORE:
- 성장여정: 스크롤해야 발견
- 플랜: Launch Plans? Recommended Plans?

AFTER:
- 성장여정: 메뉴에서 클릭 → 바로 이동
- 플랜: "런치 플랜" 메뉴 → 명확
```

## 🔄 사용자 플로우

### 시나리오 1: 성장여정 확인
```
1️⃣ 사용자가 페이지 접속
    ↓
2️⃣ 네비게이션에서 "성장여정" 클릭
    ↓
3️⃣ Growth Journey 섹션으로 스크롤
    ↓
4️⃣ 14단계 성장 과정 확인
    💭 "우리는 지금 어디쯤?"
    💭 "다음 단계는 뭐지?"
```

### 시나리오 2: 플랜 선택
```
1️⃣ 사용자가 네비게이션에서 "런치 플랜" 클릭
    ↓
2️⃣ Launch Plans 섹션으로 스크롤
    ↓
3️⃣ 3개 플랜 카드 확인
    - 스타터 플랜
    - 그로스 플랜 (가장 인기)
    - 스케일업 플랜
    ↓
4️⃣ [자세히 알아보기] 클릭
    ↓
5️⃣ 모달 팝업으로 상세 정보 확인
    ↓
6️⃣ [상담 신청] 클릭
```
