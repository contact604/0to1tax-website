### 3️⃣ FAQ 섹션 추가 (신규)

**위치**: 신뢰 섹션 ↔ CTA 섹션 사이

#### Q1. 기장이나 세무 신고도 포함되나요? ⭐ 핵심

**답변**:
```
✅ 네. Zero to One TAX는 기장부터 시작합니다.

다만 단순 기장이 아니라, 
투자자·VC·실사를 전제로 한 기장 및 재무 관리 체계를 구축합니다.
그래서 단순 기장만을 위한 서비스는 제공하지 않습니다.

기장은 투자 준비를 위한 필수 인프라이며,
우리는 그 기장 데이터가 IR에서 설명 가능하고,
실사(FDD)에서 검증 가능하도록 처음부터 설계합니다.
```

#### Q2. 일반 세무사와 뭐가 다른가요?

**답변**:
```
일반 세무사는 세금 신고와 기장 업무에 집중합니다.

Zero to One TAX는 투자 유치를 목표로 하는 스타트업에 특화:
- Big4 회계법인 출신의 투자·IPO·실사 경험
- VC가 보는 숫자 기준으로 재무 정리
- 투자 계약, 실사(FDD) 대응 체크포인트 사전 점검
- 투자 단계별(PRE-A → Series A → IPO) 맞춤 지원
```

#### Q3. 투자 전에 얼마나 미리 준비해야 하나요?

**답변**:
```
최소 3~6개월 전부터 준비하는 것이 이상적입니다.

투자 미팅 직전에 재무를 정리하면:
- 과거 데이터 보정에 시간 소요
- 구조적 문제(법인 구조, 스톡옵션 등) 해결 어려움
- 실사에서 발견되는 리스크 대응 불가
```

#### Q4. 어떤 스타트업에게 적합한가요?

**답변**:
```
PRE-A부터 Series A 투자를 준비하는 스타트업에게 가장 적합합니다.

특히 이런 경우 Zero to One TAX가 도움이 됩니다:
- VC 미팅을 앞두고 있지만 재무가 정리되지 않은 경우
- IR 자료는 있지만 숫자 근거가 부족한 경우
- 실사(FDD)를 앞두고 체크포인트가 불분명한 경우
- 법인 구조나 스톡옵션 설계가 투자에 맞지 않는 경우
```

function toggleFAQ(element) {
    const faqItem = element.parentElement;
    const isActive = faqItem.classList.contains('active');
    
    // Close all FAQ items
    document.querySelectorAll('.faq-item').forEach(item => {
        item.classList.remove('active');
    });
    
    // Open clicked item if it wasn't active
    if (!isActive) {
        faqItem.classList.add('active');
    }
}
