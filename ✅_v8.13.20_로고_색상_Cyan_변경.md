```

```

#### 2. 타겟 고객
```

연령대: 30-40대
성향: 혁신적, 도전적, 성장 지향

→ Cyan이 더 어필!
```

#### 3. 시장 차별화
```
경쟁사: 대부분 Blue 사용
세무업계: 보수적 이미지 많음

→ Cyan으로 차별화!
```

#### 4. 브랜드 메시지
```
메시지: "0에서 1을 만들어가는 여정"
가치: 동반 성장, 혁신 지원
```
