## 🔧 수정 내용

### 문제
FAQ 아코디언이 작동하지 않음
- 클릭해도 답변이 펼쳐지지 않음
- max-height 제한으로 인한 문제

### 원인
```css
/* Before */
.faq-item.active .faq-answer {
    max-height: 500px;  /* 너무 작음 */
    padding: 0 30px 25px 30px;
}
```

FAQ 답변 내용이 길어서 500px 초과 시 잘림

### 해결
```css
/* After */
.faq-item.active .faq-answer {
    max-height: 1000px;  /* 2배 증가 */
    padding: 0 30px 25px 30px;
}

/* 모바일도 동일하게 수정 */
@media (max-width: 768px) {
    .faq-item.active .faq-answer {
        max-height: 1000px;
        padding: 0 18px 18px 18px;
    }
}
```

## 🎯 FAQ 아코디언 기능

### 작동 방식
```javascript
function toggleFAQ(element) {
    const faqItem = element.parentElement;
    const isActive = faqItem.classList.contains('active');
    
    // 모든 FAQ 닫기
    document.querySelectorAll('.faq-item').forEach(item => {
        item.classList.remove('active');
    });
    
    // 클릭한 항목만 열기 (이미 열려있었다면 닫힘)
    if (!isActive) {
        faqItem.classList.add('active');
    }
}
```

### CSS 애니메이션
- **높이 전환**: max-height 0 → 1000px
- **패딩 전환**: 0 30px → 0 30px 25px 30px
- **아이콘 회전**: 0deg → 45deg
- **전환 시간**: 0.3s ease
