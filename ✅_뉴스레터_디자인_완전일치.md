## 디자인 상세

### 1️⃣ 인사말
```css
- 폰트 크기: 16px
- 색상: #333333 (검은색)
- 굵기: 400 (보통)
- 여백: 하단 5px
```

### 2️⃣ 파란색 박스 (intro-box)
```css
배경색: #e3f2fd (연한 파란색)
테두리: 왼쪽 4px solid #2196f3 (파란색)
패딩: 20px
모서리: 4px 둥글게
폰트 크기: 15px
줄 간격: 1.8
색상: #333333
```

### 3️⃣ 강조 텍스트 (박스 안)
```css
색상: #1565c0 (진한 파란색)
굵기: 600 (세미볼드)

강조된 단어:
- "0에서 1을 만들어가는 위대한 여정"
- "곁에서 항상 도움"
- "세무, 회계 서비스"
```

### 4️⃣ 링크 스타일
```css
색상: #2196f3 (파란색)
굵기: 600 (세미볼드)
폰트 크기: 15px
여백: 상단 15px (첫 번째), 8px (두 번째)
아이콘: 🚀, 👤

Hover 효과:
- 색상: #1565c0 (진한 파란색)
- 밑줄 표시
```

## 주요 변경점

### ✅ 인사말 간소화
- "안녕하세요, 0to1tax 이범기 회계사입니다."로 한 줄 정리

### ✅ 파란색 박스 추가
- 연한 파란색 배경 (#e3f2fd)
- 왼쪽 파란색 테두리 (4px)
- 적절한 패딩과 둥근 모서리
