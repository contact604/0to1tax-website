## 변경 사항

### Before
- 맨 위에 큰 카드 형태로 표시
- 구독 폼 위에 위치

### After ✨
- "최근 발행된 뉴스레터" 섹션의 맨 앞에 추가
- 다른 뉴스레터들과 동일한 형식
- NEW 배지로 최신임을 강조

```
    └── ... (기타 뉴스레터)
```

## 디자인 특징

### 1️⃣ NEW 배지
```html
<span class="bg-blue-500 text-white px-2 py-1 rounded-full">
    NEW
</span>
```
- 위치: 우측 상단
- 색상: 파란색 배경 + 흰색 텍스트
- 강조: 최신 뉴스레터임을 표시

### 2️⃣ 강조된 테두리
```css
border: border-blue-400 border-opacity-40
hover: border-opacity-70
```
- 일반 뉴스레터: 연한 파란색 (opacity-20)
- 최신 뉴스레터: 진한 파란색 (opacity-40) ⭐
- 호버: 더 진하게 (opacity-70)

### 3️⃣ 아이콘
```html
<i class="fas fa-newspaper text-blue-400"></i>
```
- 배경: 파란색 원형 (bg-opacity-30)
- 아이콘: 신문 아이콘

### 4️⃣ 추가 설명
```html
<p class="text-xs text-gray-500">
    🤔 나에게 뭐가 달라질까?
</p>
```
- 부제로 호기심 유발
