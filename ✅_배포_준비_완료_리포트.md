## 🚀 배포 방법 (2가지)

### 🌟 방법 1: Publish 탭 사용 (가장 쉬움!)

1. **Publish 탭** 클릭
2. **"Publish" 버튼** 클릭
3. 배포 완료! 🎉
4. 제공된 URL로 접속

**예상 시간**: 1분 이내

---

### 🌟 방법 2: GitHub + Netlify 연동 (자동 배포)

#### Step 1: GitHub 저장소 생성
```bash
# GitHub에서 새 저장소 생성
이름: 0to1tax-website
공개 여부: Public 또는 Private
```

#### Step 2: 파일 업로드

**옵션 A: GitHub 웹에서 직접**
1. GitHub 저장소 페이지 이동
2. "Add file" → "Upload files" 클릭
3. 모든 파일을 드래그 앤 드롭
4. Commit 메시지: `Initial commit: Upload website files`
5. "Commit changes" 클릭

**옵션 B: Git 명령어 (터미널)**
```bash
git init
git add .
git commit -m "Initial commit"
git remote add origin https://github.com/사용자명/0to1tax-website.git
git push -u origin main
```

#### Step 3: Netlify 연동
1. https://app.netlify.com 로그인
2. "Add new site" → "Import an existing project"
3. "GitHub" 선택
4. 저장소 선택: `0to1tax-website`
5. Build settings:
   - **Build command**: (비워두기)
   - **Publish directory**: `.` (루트)
6. "Deploy site" 클릭

#### Step 4: 완료! 🎉
- Netlify가 자동으로 배포
- `https://random-name.netlify.app` 형태의 URL 제공
- 이후 GitHub에 push하면 자동 재배포!

**예상 시간**: 10~15분
