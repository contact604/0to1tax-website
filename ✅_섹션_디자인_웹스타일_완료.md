## 🎯 디자인 특징

### 1️⃣ **섹션 컨테이너** (회색 박스)
- 배경색: `#f8fafc` (연한 회색)
- 테두리: `1px solid #e2e8f0`
- 모서리: `border-radius: 8px`
- 패딩: `25px`

### 2️⃣ **섹션 제목** (파란 그라디언트)
```
╔═══════════════════════════════════╗
║ ① 고용 · 출산 인센티브 강화       ║  ← 흰색 텍스트
╚═══════════════════════════════════╝
```
- 배경: 파란 그라디언트 (`#1e3a8a` → `#3b82f6`)
- 텍스트: 흰색, 굵게, 20px
- 패딩: `15px 20px`
- 모서리: `border-radius: 6px`

### 3️⃣ **콘텐츠 카드** (흰색 박스)
```
┌─────────────────────────────────┐
│ 통합고용세액공제 추징규정 삭제  │  ← 진한 파란색 제목
│ 기존 구조에서 일부 감원 시...   │  ← 회색 설명
└─────────────────────────────────┘
```
- 배경색: `#ffffff` (흰색)
- 왼쪽 테두리: `4px solid #3b82f6` (파란색 강조선)
- 모서리: `border-radius: 4px`
- 패딩: `15px 20px`
- 카드 간격: `margin-bottom: 15px`

#### 제목 스타일
- 폰트: 16px, 굵게
- 색상: `#1e3a8a` (진한 파란색)
- 여백: `margin-bottom: 8px`

#### 설명 스타일
- 폰트: 15px
- 색상: `#64748b` (회색)
- 줄 간격: `1.8`
