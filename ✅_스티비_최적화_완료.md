## 🎯 스티비 최적화 사항

### 1️⃣ **완전한 인라인 스타일**
- ❌ Before: `<style>` 태그 내부 CSS
- ✅ After: 모든 스타일을 `style="..."` 속성으로 직접 적용
- 스티비는 `<style>` 태그를 제한적으로 지원하므로 인라인 필수

### 2️⃣ **테이블 기반 레이아웃**
```html
<table width="600" cellpadding="0" cellspacing="0" border="0">
  <tr>
    <td>콘텐츠</td>
  </tr>
</table>
```
- 모든 섹션을 `<table>` 구조로 변경
- 가장 안정적인 이메일 렌더링 방식

### 3️⃣ **CSS 애니메이션 제거**
- ❌ Before: `animation: ctaBoxPulse 3s`, `@keyframes` 등
- ✅ After: 정적 스타일만 사용
- 스티비는 CSS 애니메이션을 지원하지 않음

### 4️⃣ **버튼을 테이블로 구현**
```html
<table cellpadding="0" cellspacing="0" border="0" align="center">
  <tr>
    <td style="background: #3b82f6; border-radius: 30px;">
      <a href="링크" style="display: inline-block; padding: 16px 40px;">
        버튼 텍스트
      </a>
    </td>
  </tr>
</table>
```
- 클릭 영역 정확히 보장
- 모든 이메일 클라이언트 호환
