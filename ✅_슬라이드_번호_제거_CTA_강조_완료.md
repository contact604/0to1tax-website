## 변경 사항 2: CTA 박스 강조

### 데스크탑 스타일

#### Before (차분한 디자인)
```css
.cta-section {
    background: #f8fafc;                              /* 연한 회색 */
    border: 1px solid rgba(59, 130, 246, 0.15);      /* 얇은 테두리 */
    padding: 40px 35px;
    border-radius: 12px;
    box-shadow: 0 2px 8px rgba(0, 0, 0, 0.04);       /* 은은한 그림자 */
}

.cta-section p {
    font-size: 1.2rem;
    color: var(--gray-text);                          /* 회색 텍스트 */
    font-weight: 600;
}
```

#### After (강조된 디자인) ⭐
```css
.cta-section {
    background: linear-gradient(135deg, #eff6ff 0%, #dbeafe 100%);  /* 파란 그라디언트 */
    border: 2px solid #3b82f6;                       /* 진한 파란 테두리 */
    padding: 45px 40px;                              /* 더 큰 여백 */
    border-radius: 16px;                             /* 더 둥근 모서리 */
    box-shadow: 0 8px 24px rgba(59, 130, 246, 0.15); /* 강한 파란 그림자 */
}

.cta-section p {
    font-size: 1.3rem;                               /* 더 큰 글자 */
    color: var(--primary-blue);                      /* 진한 파란색 */
    font-weight: 700;                                /* 더 굵게 */
}
```

## 세부 변경사항

### 배경
**Before:** 단색 회색 `#f8fafc`  
**After:** 그라디언트 `#eff6ff → #dbeafe` (하늘색 → 연한 파란색)

### 테두리
**Before:** `1px solid rgba(59, 130, 246, 0.15)` (연한 파란색)  
**After:** `2px solid #3b82f6` (진한 파란색, 2배 두께)

### 그림자
**Before:** `0 2px 8px rgba(0, 0, 0, 0.04)` (중립적 회색)  
**After:** `0 8px 24px rgba(59, 130, 246, 0.15)` (파란색, 3배 크기)

### 패딩
**Before:** `40px 35px`  
**After:** `45px 40px` (더 넉넉하게)

### 둥근 모서리
**Before:** `12px`  
**After:** `16px` (더 둥글게)

### 텍스트
**Before:** `1.2rem`, 회색, `font-weight: 600`  
**After:** `1.3rem`, 파란색, `font-weight: 700`

## CTA 강조 효과

### 1. 시각적 집중도 ⬆️
✅ 파란 그라디언트 배경으로 눈에 띔  
✅ 진한 파란 테두리로 영역 강조  
✅ 파란 그림자로 입체감

### 2. 텍스트 가독성 ⬆️
✅ 더 큰 글자 (1.2rem → 1.3rem)  
✅ 진한 파란색 (회색 → 파란색)  
✅ 더 굵은 글씨 (600 → 700)

### 3. 행동 유도 ⬆️
✅ 강조된 디자인으로 클릭 유도  
✅ 전문적이면서도 눈에 띄는 느낌  
✅ 신뢰감 있는 파란색 계열
