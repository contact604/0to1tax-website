## 디자인 요소

### 1️⃣ 카드 스타일
```css
- 배경: 파란색 그라디언트 (blue-900 → blue-800)
- 테두리: 2px 파란색 (border-blue-400)
- 효과: 호버 시 확대 (scale-1.02)
- 그림자: 파란색 그림자 (hover:shadow-blue-500/50)
```

### 2️⃣ 배지
```html
🆕 최신 뉴스레터
- 배경: 파란색 투명 (bg-blue-500 bg-opacity-20)
- 테두리: 파란색 (border-blue-400)
```

### 3️⃣ 3개 박스
```
① 고용·출산 인센티브 강화
② 자본시장 활성화 정책
③ 세율 원래 수준 환원
```

### 4️⃣ 2개 버튼
```
[📧 뉴스레터 전문 보기]  (메인 CTA)
- 링크: newsletter-2026-tax-reform.html
- 스타일: 파란색 그라디언트
- 효과: 호버 시 확대

[📊 슬라이드 보기]  (보조 CTA)
- 링크: 2026-tax-reform-slides.html
- 스타일: 투명 배경, 테두리
```

### 5️⃣ 발송 날짜
```
📅 2025년 1월 4일 발송
- 위치: 하단
- 스타일: 작은 텍스트, 테두리 상단
```

## 호버 효과

### 카드 전체
```css
hover:scale-[1.02]
hover:shadow-blue-500/50
transition-all duration-300
```

### 메인 버튼
```css
hover:from-blue-600 hover:to-blue-700
hover:shadow-blue-500/50
hover:scale-105
```

### 보조 버튼
```css
hover:bg-opacity-20
hover:border-opacity-50
```
