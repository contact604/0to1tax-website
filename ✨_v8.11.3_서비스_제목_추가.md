## 🎨 주요 변경사항

### 1. 서비스 제목 추가

**위치**: 로고 이미지 바로 아래

**스타일**:
```css
.service-title {
    font-size: 28px;           /* 크게 */
    font-weight: 700;          /* 굵게 */
    color: #FFD200;            /* 노란색 (눈에 띄게) */
    margin-bottom: 30px;
    letter-spacing: -0.5px;
}
```

**모바일**:
```css
.service-title {
    font-size: 22px;           /* 모바일에서도 크게 */
    margin-bottom: 25px;
}
```
