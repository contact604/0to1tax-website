# 🌐 0to1tax.com 도메인 연결 - 초간단 가이드

## 현재 상황
- ✅ 웹사이트 배포 완료
- ✅ 도메인 구매 완료 (0to1tax.com)
- ❌ 도메인과 웹사이트가 연결 안 됨

---

## 🎯 1단계: 실제 사이트 URL 확인하기

**Publish 탭**에서 배포된 실제 URL을 확인하세요:
```
예시: https://iadkczem.gensparkspace.com
```

이 URL로 접속하면 사이트가 보입니다! 👍

---

## 🎯 2단계: 도메인 연결하기 (2가지 방법)

### 방법 A: Squarespace DNS 설정 (추천) ⭐

**장점**: 간단하고 빠름 (10분)

#### 단계:
1. **Squarespace 로그인**
2. **도메인** → **0to1tax.com** → **DNS 설정**
3. **CNAME 레코드 추가**:
   ```
   Host: www
   Points to: iadkczem.gensparkspace.com
   TTL: Auto 또는 3600
   ```
4. **A 레코드 추가** (루트 도메인용):
   ```
   Host: @
   Points to: [배포된 서버 IP 주소]
   TTL: Auto 또는 3600
   ```

#### 결과:
- ⏱️ 15분~2시간 후 작동
- ✅ www.0to1tax.com 접속 가능
- ⚠️ SSL (https) 문제 있을 수 있음

---

### 방법 B: Cloudflare 사용 (권장) ⭐⭐⭐

**장점**: 
- ✅ 무료 SSL (https) 자동 발급
- ✅ 빠른 속도 (CDN)
- ✅ DDoS 보호
- ✅ 전문적인 관리

#### 단계:
1. **Cloudflare 가입**: https://dash.cloudflare.com/sign-up
2. **"Add a Site"** 클릭 → **0to1tax.com** 입력
3. **Free 플랜** 선택
4. DNS 레코드 확인:
   ```
   Type: CNAME
   Name: www
   Target: iadkczem.gensparkspace.com
   Proxy: ON (주황색 구름)
   ```
5. Cloudflare가 제공하는 **네임서버 2개** 복사
6. **Squarespace DNS 설정**으로 가서 네임서버 변경
7. 저장하고 대기 (15분~24시간)

#### 결과:
- ⏱️ 15분~24시간 후 작동
- ✅ https://www.0to1tax.com 완벽 작동
- ✅ 보안 + 속도 향상

## 🎯 3단계: 구글 검색 노출 준비

도메인 연결 후 해야 할 일:

### A. Google Search Console 등록 (필수!)
1. https://search.google.com/search-console 접속
2. **속성 추가** → **0to1tax.com** 입력
3. **도메인 소유권 확인**
4. **sitemap.xml 제출**: https://0to1tax.com/sitemap.xml

### B. Naver 웹마스터도구 등록
1. https://searchadvisor.naver.com 접속
2. 사이트 추가
3. 사이트맵 제출

### C. 실제 콘텐츠 추가
- 블로그 글 작성 (admin.html에서)
- 정기적으로 업데이트
- 고객 리뷰 추가
