---

## 📝 체크리스트 (우선순위 순)

### 🔴 긴급 (지금 바로!) - 10분

- [ ] **1. Publish 탭에서 배포된 URL 확인하기**
  - 예시: `https://iadkczem.gensparkspace.com`
  - 이 URL로 접속해서 사이트가 잘 보이는지 확인

- [ ] **2. Google Search Console 가입**
  - 🔗 https://search.google.com/search-console
  - 구글 계정으로 로그인
  - "속성 추가" 클릭

- [ ] **3. 사이트 URL 등록**
  - 배포된 URL 또는 0to1tax.com 입력
  - 소유권 확인 (HTML 파일 방법이 가장 쉬움)

---

### 🟠 중요 (오늘 안에!) - 20분

- [ ] **4. sitemap.xml 제출**
  - Search Console → "Sitemaps" 메뉴
  - `sitemap.xml` 입력 후 제출

- [ ] **5. Naver 웹마스터도구 등록**
  - 🔗 https://searchadvisor.naver.com
  - 사이트 추가
  - sitemap.xml 제출

- [ ] **6. 첫 블로그 글 작성**
  - admin.html 접속
  - 주제 예시:
    - "스타트업 법인 설립 완벽 가이드"
    - "투자 유치 후 꼭 해야 할 세무 절차"
    - "스톡옵션 세금 계산 방법"

---

### 🟡 보통 (이번 주 안에!) - 1시간

- [ ] **7. 도메인 연결 (선택사항)**
  - 방법 A: Cloudflare 사용 (추천)
    - 가이드: `CLOUDFLARE_SETUP_GUIDE.md` 참고
  - 방법 B: Squarespace DNS 설정
    - 가이드: `DOMAIN_ROOT_SETUP.md` 참고

- [ ] **8. Google Analytics 설치 (선택사항)**
  - 🔗 https://analytics.google.com
  - 추적 코드 발급
  - index.html에 추가

- [ ] **9. 블로그 글 5개 작성**
  - 주 1~2개씩 작성
  - 고객이 궁금해할 내용 위주

---

### 🟢 좋으면 좋음 (여유 있을 때) - 2시간

- [ ] **10. FAQ 섹션 업데이트**
  - 실제 고객 문의 내용 추가
  - 상세한 답변 작성

- [ ] **11. 고객 리뷰/사례 추가**
  - 실제 고객 후기 수집
  - 성공 사례 작성

- [ ] **12. SNS 계정 연결**
  - Facebook, Instagram 링크 실제 계정으로 변경
  - 블로그 글 SNS에 공유

- [ ] **13. 카카오톡 채널 연결**
