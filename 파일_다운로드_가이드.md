## 🚀 빠른 업로드 방법

### 방법 1: GitHub 웹 인터페이스 (가장 쉬움!)

**1. GitHub 저장소 페이지로:**
```
https://github.com/yourusername/0to1tax-website
```

**2. "Add file" 클릭:**
- **Upload files** 선택

**3. 파일 업로드:**
- 파일을 드래그 앤 드롭
- 또는 "choose your files" 클릭해서 선택

**4. 커밋 메시지:**
```
Initial commit: Upload website files
```

**5. "Commit changes" 클릭!**

---

## 📦 파일 준비 방법

### 현재 환경에서 파일 다운로드:

**각 파일별로:**
1. 파일 열기
2. 내용 전체 복사 (Ctrl + A, Ctrl + C)
3. 로컬에 텍스트 파일로 저장
4. 파일명 정확히 지정 (예: index.html)

**또는**:

프로젝트에 다운로드 기능이 있다면 사용!
