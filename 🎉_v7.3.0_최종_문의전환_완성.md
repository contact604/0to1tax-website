**변경 후**: 핵심 3개 문장만 남김 ✅

```
1. 이범기 대표 회계사는
   대형 회계법인(Big4급) 감사본부와 IPO·투자·실사(Due Diligence) 현장에서
   투자자가 실제로 무엇을 보고 판단하는지를 경험해 왔습니다.

2. 그래서 단순히 재무제표를 정리하는 것이 아니라,
   투자자가 질문하는 구조 자체를 먼저 설계합니다.

3. Zero to One TAX는
   기장부터 IR·실사까지 이어지는 재무 구조를
   처음부터 투자 관점에서 설계합니다.
```

**효과**:
- ❌ 중복 제거 (VC 질문, 실사 팀 등 세부 설명 삭제)
- ❌ 방어적 문구 제거 (하이라이트 카드 삭제)
- ✅ 핵심 메시지만 전달 (읽기 쉽고 명확)

### 3️⃣ 문의 폼 노출 방식 변경 (핵심 변경)

**변경 전**: 문의 폼 항상 노출 (모든 방문자에게)

**변경 후**: 접기 방식 → CTA 클릭 시에만 열림 ✅

**구조**:
```
[기본 상태]
📍 대표 소개 → 법인 소개 → [💬 투자 준비 문의하기] 버튼
                                    ↓ (클릭 시)
[열림 상태]
📍 대표 소개 → 법인 소개 → [💬 투자 준비 문의하기] 버튼
                              ↓
                       [문의 폼 펼쳐짐]
```

**CSS 애니메이션**:
- `max-height: 0` → `max-height: 2000px` (부드러운 펼침)
- `opacity: 0` → `opacity: 1` (페이드 인)
- `transition: all 0.5s ease` (0.5초 애니메이션)

**JavaScript 동작**:
```javascript
function toggleContactForm() {
    const wrapper = document.getElementById('contactFormWrapper');
    wrapper.classList.toggle('open');
    
    // 펼칠 때 부드럽게 스크롤
    if (wrapper.classList.contains('open')) {
        setTimeout(() => {
            wrapper.scrollIntoView({ behavior: 'smooth', block: 'nearest' });
        }, 100);
    }
}
```

**효과**:
- ✅ 충분히 읽고 확신한 사람만 문의 폼 열람
- ✅ 문의의 질(quality) 향상
- ✅ 페이지 밀도 감소 (읽기 편함)
