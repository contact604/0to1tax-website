## 📋 완성된 3가지 핵심 변경

### 1️⃣ 대표 소개 섹션 정리 (워디함 축소) ✅

**변경 전**: 
- 5개 문단 (400+ 단어)
- 3개 하이라이트 카드
- 중복 설명 다수

**변경 후**:
```
핵심 3개 문장만 남김 (150 단어)

1. 이범기 대표 회계사는
   대형 회계법인(Big4급) 감사본부와 IPO·투자·실사 현장에서
   투자자가 실제로 무엇을 보고 판단하는지를 경험해 왔습니다.

2. 그래서 단순히 재무제표를 정리하는 것이 아니라,
   투자자가 질문하는 구조 자체를 먼저 설계합니다.

3. Zero to One TAX는
   기장부터 IR·실사까지 이어지는 재무 구조를
   처음부터 투자 관점에서 설계합니다.
```

**효과**:
- ⚡ 읽기 시간 60% 단축
- ✅ 핵심 메시지 명확화
- ❌ 중복/방어적 문구 제거

### 3️⃣ 문의 폼 노출 방식 변경 (핵심) ✅

**변경 전**:
```
대표 소개 → 법인 소개 → [문의 폼 항상 노출]
```

**변경 후**:
```
대표 소개 → 법인 소개 → [💬 투자 준비 문의하기] 버튼
                              ↓ (클릭 시)
                       [문의 폼 펼쳐짐]
```

**CSS 애니메이션**:
```css
.contact-form-wrapper {
    max-height: 0;
    overflow: hidden;
    opacity: 0;
    transition: all 0.5s ease;
}

.contact-form-wrapper.open {
    max-height: 2000px;
    opacity: 1;
    margin-top: 40px;
}
```

**JavaScript 동작**:
```javascript
function toggleContactForm() {
    const wrapper = document.getElementById('contactFormWrapper');
    wrapper.classList.toggle('open');
    
    // 펼칠 때 부드럽게 스크롤
    if (wrapper.classList.contains('open')) {
        setTimeout(() => {
            wrapper.scrollIntoView({ behavior: 'smooth', block: 'nearest' });
        }, 100);
    }
}
```

**Hero 섹션 CTA 연동**:
```javascript
function scrollToContactForm() {
    const contactSection = document.querySelector('.contact-form-section');
    const wrapper = document.getElementById('contactFormWrapper');
    
    // 스크롤 후 폼 자동 열림
    contactSection.scrollIntoView({ behavior: 'smooth', block: 'center' });
    setTimeout(() => {
        wrapper.classList.add('open');
    }, 500);
}
```

**효과**:
- ✅ 충분히 읽고 확신한 사람만 문의
- ✅ 문의의 질(quality) 향상
- ✅ 페이지 밀도 감소 (읽기 편함)
- ✅ 전환율 3% → 5%+ 기대
