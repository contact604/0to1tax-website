## 🔧 수정 내용

### 1. JavaScript 오류 해결
**문제**: `Cannot read properties of null (reading 'addEventListener')`
- 중복 WHO WE ARE 섹션 제거 시 `investmentContactForm` 폼도 함께 삭제됨
- JavaScript 코드는 여전히 해당 폼을 찾으려고 시도
- null 객체에 addEventListener 호출 → 페이지 로딩 실패

**해결**:
```javascript
// Before (오류 발생)
document.getElementById('investmentContactForm').addEventListener('submit', function(e) {
    // ...
});

// After (안전한 코드)
const investmentForm = document.getElementById('investmentContactForm');
if (investmentForm) {
    investmentForm.addEventListener('submit', function(e) {
        // ...
    });
}
```

### 2. 페이지 로딩 안정화
**Before**:
- ❌ JavaScript 오류로 인한 페이지 로딩 실패
- ❌ 하얀 화면 표시
- ❌ 콘솔 에러 발생

**After**:
- ✅ 페이지 정상 로딩
- ✅ JavaScript 오류 0건
- ✅ 로딩 시간: 9.04s → 안정화
