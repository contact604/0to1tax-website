**Before**:
```html

```

**CSS (Before)**:
```css
.faq-icon {
    font-size: 32px;
    font-weight: 300;
}

```


### 1. 직관성 향상

| **인지속도** | 2초 | <1초 | +100% |

### 2. 사용자 인터랙션

**Before**:
- 😕 "`+` 가 무슨 의미지?"
- 😕 "클릭하면 뭐가 나오나?"
- 😕 45도 회전 → × (닫기 의미? 혼란)

**After**:
- 😊 "▼ 화살표 → 아래로 펼쳐지겠구나"
- 😊 클릭 후 ▲ → "위로 접히겠구나"
- 😊 직관적인 상태 표시

### 3. 표준 UI 패턴

**일반적인 아코디언 아이콘**:
- ✅ ▼/▲ (화살표) - 가장 보편적
- ✅ >/∨ (꺾쇠)
- ⚠️ +/− (플러스/마이너스) - 덜 직관적
- ❌ +/× (플러스/엑스) - 혼란스러움

## 🔄 애니메이션 개선

### 회전 각도

**Before (45도)**:
```
+ (닫힘)
 ↓ 45° 회전
× (열림) ← 닫기처럼 보임 (혼란)
```

**After (180도)**:
```
▼ (닫힘) ← 펼치기
 ↓ 180° 회전
▲ (열림) ← 접기
```

### 애니메이션 속도
```css
transition: transform 0.3s ease;
```
- ✅ 0.3초 - 부드럽고 자연스러움
- ✅ ease - 가속/감속 자연스러움
