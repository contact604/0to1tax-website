## 🔍 Before vs After

### CSS 변경

**Before**:
```css
.faq-answer {
    max-height: 0;
    overflow: hidden;
    transition: max-height 0.3s ease;
    padding: 0 30px;
}

.faq-item.active .faq-answer {
    max-height: 1000px;
    padding: 0 30px 25px 30px;
}
```

**After**:
```css
.faq-answer {
    max-height: 0;
    overflow: hidden;
    transition: max-height 0.3s ease, padding 0.3s ease;
    padding: 0 30px;
    opacity: 0;  /* ← 추가: 완전히 숨김 */
}

.faq-item.active .faq-answer {
    max-height: 1000px;
    padding: 0 30px 25px 30px;
    opacity: 1;  /* ← 추가: 페이드 인 */
}
```

## 🎯 JavaScript 작동

### toggleFAQ 함수 (변경 없음)
```javascript
function toggleFAQ(element) {
    const faqItem = element.parentElement;
    const isActive = faqItem.classList.contains('active');
    
    // 모든 FAQ 닫기
    document.querySelectorAll('.faq-item').forEach(item => {
        item.classList.remove('active');
    });
    
    // 클릭한 항목이 닫혀있었다면 열기
    if (!isActive) {
        faqItem.classList.add('active');
    }
}
```

### 동작 원리
1. 사용자가 질문 클릭
2. 모든 `.faq-item`에서 `active` 클래스 제거
3. 클릭한 항목에만 `active` 클래스 추가
4. CSS가 자동으로 애니메이션 적용
