## 🔍 문제 분석

### 문제점

문의하기 섹션의 배경이 그라데이션으로 설정되어 있어서:
- 좌우에 흰색 여백처럼 보이는 부분 발생
- 시각적으로 섹션이 좁아 보임
- 배경 색상이 불균일하게 보임

### 원인

```css
background: linear-gradient(135deg, #f6f9ff 0%, #ffffff 100%);
```
→ 그라데이션이 좌상단 파란색(`#f6f9ff`)에서 우하단 흰색(`#ffffff`)으로 변하면서
→ 우측 하단이 흰색으로 보여 여백처럼 느껴짐

---

## ✅ 해결 방법

### 배경 스타일 변경

**Before**:
```css
background: linear-gradient(135deg, #f6f9ff 0%, #ffffff 100%);
```

**After**:
```css
background: #f6f9ff;
```

### 변경 이유

1. **일관성**: 전체 섹션이 동일한 색상
2. **깔끔함**: 그라데이션 제거로 단순하고 명확
3. **여백 제거**: 흰색 부분이 없어서 여백처럼 보이지 않음
4. **브랜드 색상 유지**: Zero to One TAX 브랜드 색상(`#f6f9ff`) 유지

---

```
│        여백 없이 깔끔함          │

```

## 🎨 색상 정보

### 사용된 색상

- **섹션 배경**: `#f6f9ff` (연한 파란색)
  - Zero to One TAX 브랜드 색상
  - 부드럽고 전문적인 느낌
  - 눈에 피로하지 않음

- **폼 배경**: `white` (흰색)
  - 명확한 입력 영역
  - 섹션 배경과 명확한 대비
  - 깔끔한 폼 디자인

### 시각적 계층

```
#f6f9ff (섹션 배경)
  ↓
  └─ white (폼 배경)
       ↓
       └─ 입력 필드
```
