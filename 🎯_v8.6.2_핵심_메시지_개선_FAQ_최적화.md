## 3️⃣ FAQ 아코디언 구조 확인

### ✅ 현재 FAQ 상태

FAQ는 이미 **완벽한 아코디언 구조**로 구현되어 있습니다!

#### 🎨 CSS 구조 (Line 1069-1127)

```css
.faq-item {
    /* 기본 카드 스타일 */
}

.faq-answer {
    max-height: 0;           /* 기본: 닫힘 */
    overflow: hidden;
    transition: max-height 0.3s ease;
}

.faq-item.active .faq-answer {
    max-height: 500px;       /* 열림 */
    padding: 0 30px 25px 30px;
}
```

#### ⚙️ JavaScript 토글 함수 (Line 2861-2874)

```javascript
function toggleFAQ(element) {
    const faqItem = element.parentElement;
    const isActive = faqItem.classList.contains('active');
    
    // 모든 FAQ 닫기
    document.querySelectorAll('.faq-item').forEach(item => {
        item.classList.remove('active');
    });
    
    // 클릭한 항목만 열기 (토글)
    if (!isActive) {
        faqItem.classList.add('active');
    }
}
```

### ✅ 작동 방식

1. **기본 상태**: 모든 FAQ 항목 닫힘 ✅
2. **클릭 시**: 해당 항목만 펼쳐짐 ✅
3. **다른 항목 클릭**: 이전 항목 닫히고 새 항목 열림 ✅
4. **같은 항목 재클릭**: 닫힘 ✅

### 🎨 시각적 효과

- **닫힘 표시**: `+` 아이콘
- **열림 표시**: `+` 아이콘 45도 회전 (✕ 모양)
- **호버 효과**: 연한 블루 배경
