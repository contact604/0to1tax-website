## 🔍 문제점

사용자가 FAQ를 볼 때 **토글 기능이 보이지 않는다**고 피드백:
- CSS `::after`로 `+` 아이콘을 추가했지만 일부 환경에서 표시 안됨
- 구조적으로 불명확
- 클릭 가능한 영역인지 인지 어려움

---

## ✅ 해결 방법

### 1️⃣ HTML 구조 개선

**Before**:
```html
<div class="faq-question" onclick="toggleFAQ(this)">
    <span>기장이나 세무 신고도 포함되나요?</span>
</div>
```

**After**:
```html
<div class="faq-question" onclick="toggleFAQ(this)">
    <span class="faq-q-text">기장이나 세무 신고도 포함되나요?</span>
    <span class="faq-icon">+</span>
</div>
```

### 2️⃣ CSS 개선

**Before**: `::after`로 가상 요소 사용
```css
.faq-question::after {
    content: '+';
    font-size: 28px;
    color: var(--primary-blue);
}

.faq-item.active .faq-question::after {
    transform: rotate(45deg);
}
```

**After**: 실제 HTML 요소에 스타일 적용
```css
.faq-icon {
    font-size: 32px;
    color: var(--primary-blue);
    font-weight: 300;
    line-height: 1;
    transition: transform 0.3s ease;
    min-width: 32px;
    text-align: center;
}

.faq-item.active .faq-icon {
    transform: rotate(45deg);
}
```

### 3️⃣ 모바일 최적화

```css
@media (max-width: 768px) {
    .faq-icon {
        font-size: 28px;
        min-width: 28px;
    }
}
```

## 🎨 작동 방식

### 📱 사용자 경험

```
[초기 상태 - 모든 FAQ 닫힘]
❓ 기장이나 세무 신고도 포함되나요?          +
❓ 일반 세무사·기장 서비스와 무엇이 다른가요?  +
❓ 아직 매출이 크지 않아도 문의해도 되나요?     +
...

[첫 번째 FAQ 클릭]
❓ 기장이나 세무 신고도 포함되나요?          ×
   📝 네, 기장과 세무 신고부터 시작합니다.
   다만 단순 장부 관리가 아니라...
❓ 일반 세무사·기장 서비스와 무엇이 다른가요?  +
❓ 아직 매출이 크지 않아도 문의해도 되나요?     +
...

[다른 FAQ 클릭]
❓ 기장이나 세무 신고도 포함되나요?          +  ← 자동 닫힘
❓ 일반 세무사·기장 서비스와 무엇이 다른가요?  ×  ← 새로 열림
   📝 일반 세무는 신고·비용 처리에 초점입니다...
❓ 아직 매출이 크지 않아도 문의해도 되나요?     +
...
```

### 🎯 핵심 기능

1. **기본 닫힘**: 모든 FAQ가 닫힌 상태로 시작
2. **클릭 열기**: 관심있는 질문 클릭 시 답변 펼쳐짐
3. **아코디언 효과**: 한 번에 하나의 FAQ만 열림 (다른 건 자동 닫힘)
4. **아이콘 회전**: `+` → `×` (45도 회전)
5. **부드러운 애니메이션**: 0.3초 슬라이드 효과
