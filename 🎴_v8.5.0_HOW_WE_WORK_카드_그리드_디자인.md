### 3. CSS 스타일 상세

#### 카드 기본 스타일
```css
.work-card {
    background: white;
    border-radius: 20px;
    padding: 40px;
    box-shadow: 0 4px 20px rgba(0, 0, 0, 0.08);
    border: 2px solid rgba(0, 102, 255, 0.1);
    transition: all 0.3s ease;
    position: relative;
    overflow: hidden;
}
```

#### 호버 효과
```css
.work-card:hover {
    transform: translateY(-5px);
    box-shadow: 0 12px 40px rgba(0, 102, 255, 0.15);
    border-color: rgba(0, 102, 255, 0.3);
}
```

**효과**:
- 위로 5px 이동
- 그림자 더 진하게 (0.08 → 0.15)
- 테두리 더 진하게 (0.1 → 0.3)

---

#### 번호 뱃지
```css
position: absolute;
top: 20px;
right: 20px;
width: 50px;
height: 50px;
background: linear-gradient(135deg, rgba(0, 102, 255, 0.1) 0%, rgba(0, 102, 255, 0.05) 100%);
border-radius: 50%;
font-size: 24px;
font-weight: 900;
color: var(--primary-blue);
```

**특징**:
- 우측 상단 절대 위치
- 그라데이션 배경 (반투명 파란색)
- 굵은 숫자
