### **2. /insights/ 디렉토리 (투자 전문 콘텐츠)** ✅

#### 콘텐츠 기준
- **주제**: 투자 / IR / 실사(FDD) / 재무 구조 / IFRS
- **톤**: 판단을 돕는 가이드 (설명형 블로그 ❌)
- **대상**: PRE-A ~ Series A 투자 준비 중인 대표급

#### 랜딩 연계
✅ 메인 랜딩 하단에 **3개만 노출**  
✅ "투자 준비 인사이트" 명칭 (블로그 표현 제외)  
✅ 각 글 하단 CTA: **"💬 투자 준비 진단 요청하기"** → 랜딩 #contact

#### 디자인 특징
- **색상**: 파란색 그라데이션 (`#0066FF ~ #0052CC`)
- **태그**: IR·미팅, 실사 대응, PRE-A 준비, Series A, 법인·세무 구조
- **필터**: 전체, PRE-A, IR, 실사, Series A, 법인·세무 구조

#### 샘플 콘텐츠 (6개)
| 제목 | 태그 | 상태 |
|------|------|------|
| VC 미팅에서 꼭 준비해야 할 재무 숫자는? | IR·미팅 | ✅ 완성 |
| 투자 실사(FDD)에서 문제되는 재무 구조는? | 실사 대응 | 🟡 템플릿 |
| 법인 설립 단계부터 투자 관점으로 설계해야 하는 이유 | 법인·세무 구조 | 🟡 템플릿 |
| PRE-A 투자 준비, 언제부터 시작해야 할까? | PRE-A 준비 | 🟡 템플릿 |
| Series A 투자 계약서, 재무적으로 체크해야 할 조항은? | Series A | 🟡 템플릿 |
| 스톡옵션 설계, 세무 리스크 없이 진행하는 방법 | 법인·세무 구조 | 🟡 템플릿 |

### **3. /blog/ 디렉토리 (일반 회계·보조금 블로그)** ✅

#### 콘텐츠 기준
- **주제**: 보조금, 세무 상식, 회계 일반, 스타트업 팁
- **목적**: 검색 유입(SEO) 및 정보 제공
- **대상**: 초기 스타트업, 일반 사업자 포함

#### 중요한 제한
❌ 메인 랜딩 하단에 노출하지 않음  
❌ 투자 전환용 CTA 강하게 넣지 않음  
✅ Footer에서만 접근 가능 (보조 메뉴)

#### 디자인 특징
- **색상**: 녹색 그라데이션 (`#4CAF50 ~ #388E3C`)
- **태그**: 보조금, 세무 상식, 회계 일반, 스타트업 팁
- **필터**: 전체, 보조금, 세무 상식, 회계 일반, 스타트업 팁

#### 샘플 콘텐츠 (6개)
| 제목 | 태그 | 상태 |
|------|------|------|
| 2025년 스타트업이 신청할 수 있는 정부 보조금 정리 | 보조금 | 🟡 템플릿 |
| 연말정산 앞두고 사업자가 꼭 체크해야 할 세무 항목 | 세무 상식 | 🟡 템플릿 |
| 스타트업 대표가 알아야 할 회계 기본 용어 정리 | 회계 일반 | 🟡 템플릿 |
| 초기 스타트업, 기장은 언제부터 시작해야 할까? | 스타트업 팁 | 🟡 템플릿 |
| 부가가치세 신고, 처음 하는 사업자를 위한 가이드 | 세무 상식 | 🟡 템플릿 |

## 🎯 역할 분리 원칙

### **랜딩 페이지 (/)** - 문의 전환 전용
```
Hero → 문제 인식 → 해결책 → 서비스 → 비용 → CEO 소개 → 문의 CTA → FAQ → 인사이트(3개)
```
- **목적**: 설득과 문의 전환
- **CTA**: "💬 투자 준비 문의하기"
- **전환율**: 높은 전환 압력 유지

### **/insights/** - 전환 보조 + 신뢰 형성
```
목록 페이지 → 콘텐츠 소비 → CTA: "투자 준비 진단 요청하기" → 랜딩 문의 폼
```
- **목적**: 전문성 인지, 신뢰 형성, 전환 보조
- **CTA**: 강한 전환 유도 (랜딩 #contact 연결)
- **타깃**: PRE-A ~ Series A 투자 준비 대표급

### **/blog/** - 검색 유입 전용
```
검색 유입 → 콘텐츠 소비 → (약한 CTA) → 브랜드 인지
```
- **목적**: SEO 트래픽 확보, 브랜드 인지도
- **CTA**: 약한 전환 압력 (정보 제공 중심)
- **타깃**: 초기 스타트업, 일반 사업자

## 📈 전환 흐름 설계

### **검색 유입 → 전환**

#### /insights 경로 (투자 준비 대상)
```
Google: "VC 미팅 재무 숫자"
    ↓
/insights/ir-numbers.html
    ↓
콘텐츠 소비 (전문성 인지)
    ↓
하단 CTA: "💬 투자 준비 진단 요청하기"
    ↓
index.html#contact (랜딩 문의 폼)
    ↓
문의 제출 → 1영업일 내 회신
```

#### /blog 경로 (일반 정보 탐색)
```
Naver: "스타트업 보조금"
    ↓
/blog/startup-subsidy-guide.html
    ↓
콘텐츠 소비 (정보 획득)
    ↓
약한 CTA: "더 자세한 상담이 필요하시면 문의하기"
    ↓
(선택적) 랜딩 방문 or 이탈
```

### **랜딩 직접 유입**
```
Direct / SNS / 광고
    ↓
index.html (랜딩 페이지)
    ↓
전환 흐름 스크롤
    ↓
하단 인사이트 섹션 도달
    ↓
관심 있는 콘텐츠 클릭 → /insights/
    ↓
콘텐츠 소비 → CTA → 다시 랜딩 문의 폼
```
