# 💬 v8.7.8 문의하기 CTA 톤 개선 완료

## 📋 작업 요약

**버전**: v8.7.8  
**날짜**: 2026-01-10  
**작업**: 문의하기 톤 개선 → 전문가에게 편하게 묻는 '문의' 느낌 강화

---

## 🎯 사용자 요청

> "문의하기 버튼은 '확인'보다는 전문가에게 편하게 묻는 '문의' 톤을 유지하고 싶습니다."

### 요청사항 정리

1. **버튼 문구**: "재무 상태 문의하기" (1순위)
2. **버튼 하단 보조 문구**: 클릭 부담 낮추기
   - "간단한 문의만 주셔도 충분합니다"
   - "계약 전 단계의 문의도 환영합니다"
3. **PDF 안내**: 메인 → 보조 안내로 변경
   - "문의 주시면 상담 참고용으로 서비스 소개서(PDF)를 보내드립니다"
   - "문의 후 서비스 개요 자료를 함께 전달드립니다"
4. **톤**: 마케팅 느낌 없이, 상담 참고 자료 제공 정도로 가볍게

---

## ✅ Before vs After

### 1️⃣ 섹션 제목

**Before**:
```
재무 상태 점검 문의하기
```
→ 무겁고 공식적, "점검"이라는 단어가 부담감

**After**:
```
재무 상태 문의하기
```
→ 간결하고 부드러움, 가벼운 문의 느낌

---

### 2️⃣ 서브타이틀

**Before**:
```
문의 주시면 상담 전 참고 자료로
Zero to One TAX 서비스 소개서(PDF)를 자동 발송해 드립니다.
```
→ PDF 발송을 메인 메시지로 강조 (마케팅 느낌)

**After**:
```
재무·세무에 대해 편하게 문의해 주세요.
간단한 질문도 환영합니다.
```
→ 문의를 환영한다는 부드러운 톤

---

### 3️⃣ 이메일 필드 안내

**Before**:
```
💡 이메일로 서비스 소개서 PDF를 자동 발송해 드립니다
```
→ 기능 설명

**After**:
```
💡 문의 후 상담 참고용 서비스 소개서(PDF)를 함께 보내드립니다
```
→ 보조 자료 제공이라는 부드러운 표현

---

### 4️⃣ 제출 버튼

**Before**:
```
📩 문의하기 및 서비스 소개서 받기
```
→ 기능 나열, 너무 길고 판매 느낌

**After**:
```
재무 상태 문의하기
```
→ 심플하고 명확, 문의에 집중

---

### 5️⃣ 버튼 하단 보조 문구 (신규 추가)

**Before**: (없음)

**After**:
```
간단한 문의만 주셔도 충분합니다
```
→ 심리적 장벽 제거, 부담 최소화

---

### 6️⃣ PDF 안내 문구

**Before**:
```
문의 주시면 즉시 이메일로 서비스 소개서 PDF를 보내드립니다.
상담 전 참고하시면 Zero to One TAX 서비스를 더욱 잘 이해하실 수 있습니다.
1영업일 이내 담당자가 연락드립니다.
```
→ PDF 발송을 메인으로 강조

**After**:
```
💡 문의 주시면 상담 참고용으로 서비스 소개서(PDF)를 보내드립니다.
1영업일 이내 담당자가 연락드립니다.
```
→ 보조 안내로 이동, 경계선으로 분리, 작은 글씨, 가벼운 톤

---

### 7️⃣ Hero CTA 버튼

**Before**:
```
💬 우리 회사 재무 상태 점검하기
```

**After**:
```
💬 재무 상태 문의하기
```
→ 간결하고 부담 없는 문구
