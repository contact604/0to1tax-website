## 📋 섹션 1: 투자 단계별 제공 서비스

### 위치
투자 여정 섹션 이후 → 신뢰 블록 이전

### 구조
```
┌─────────────────────────────────┐
│  투자 단계별 제공 서비스           │
├─────────────────────────────────┤
│  [PRE-A 준비]                    │
│  - 요약 (2문장)                   │
│  - 제공 서비스 4개                │
├─────────────────────────────────┤
│  [Series A 대응]                 │
│  - 요약 (2문장)                   │
│  - 제공 서비스 4개                │
├─────────────────────────────────┤
│  [투자 이후 관리]                 │
│  - 요약 (2문장)                   │
│  - 제공 서비스 4개                │
└─────────────────────────────────┘
```

### 내용

#### PRE-A 준비
**요약**:
```
투자자가 보는 숫자 기준으로 처음부터 기장·재무 데이터를 설계합니다.
법인 구조, 주주 구성, IR 재무 데이터까지 투자 관점에서 정리합니다.
```

**제공 서비스**:
- ✓ 투자 기준 기장·재무 관리 체계
- ✓ 법인 구조 초기 세팅
- ✓ VC 관점 재무제표 체계
- ✓ 스톡옵션 설계 검토

#### Series A 대응
**요약**:
```
IR 숫자 근거를 명확히 하고, 실사(Due Diligence) 체크포인트를 사전 정리합니다.
투자 계약서의 재무 조건과 세무 구조 충돌을 사전 방지합니다.
```

**제공 서비스**:
- ✓ IR 재무 숫자 검증
- ✓ 실사(FDD) 대응 자료 준비
- ✓ 투자 계약 재무 조건 검토
- ✓ 세무 리스크 사전 정리

#### 투자 이후 관리
**요약**:
```
투자 이후 외부감사 대응, 이사회 재무 보고, 지배구조 관리까지 지원합니다.
Series B·C 또는 IPO 준비를 위한 재무 체계를 유지합니다.
```

**제공 서비스**:
- ✓ 외부감사 대응 체계
- ✓ 이사회 재무 보고
- ✓ 지배구조 세무 관리
- ✓ IPO 로드맵 재무 설계

## 📋 섹션 2: 서비스 비용 안내 (가이드)

### 위치
서비스 요약 섹션 이후 → 신뢰 블록 이전

### 구조
```
┌─────────────────────────────────┐
│  서비스 비용 안내 (가이드)         │
├─────────────────────────────────┤
│  💡 비용 책정 기준                │
│  (투자 단계·규모·복잡도 설명)      │
├─────────────────────────────────┤
│  [PRE-A 준비 단계]                │
│  월 50~100만 원대                 │
│  - 포함 내용 / 별도 협의 사항     │
├─────────────────────────────────┤
│  [Series A 대응]                 │
│  프로젝트별 300~800만 원대         │
│  - 포함 내용 / 병행 시 추가       │
├─────────────────────────────────┤
│  [투자 이후 관리]                 │
│  월 150~300만 원대                │
│  - 포함 내용 / 조정 기준          │
├─────────────────────────────────┤
│  ⚠️ 주의사항 (참고용 가이드)       │
│  목적: 서비스 적합성 판단          │
└─────────────────────────────────┘
```

### 내용

#### 비용 책정 기준
```
투자 단계, 법인 규모, 거래 복잡도, IR·실사 일정에 따라
맞춤형으로 산정되며, 정확한 견적은 상담 후 제공됩니다.
```

#### PRE-A 준비 단계
**가격 범위**: 월 50~100만 원대

**포함 내용**:
- 기장·재무 데이터 관리
- 투자 기준 재무 체계 구축

**별도 협의**:
- 법인 구조, 스톡옵션 설계, IR 준비 등

#### Series A 대응 (IR·실사)
**가격 범위**: 프로젝트별 300~800만 원대

**포함 내용**:
- IR 재무 검증
- 실사 대응
- 투자 계약 재무 조건 검토

**병행 시**:
- 기장 서비스 + 프로젝트 비용 추가

#### 투자 이후 관리
**가격 범위**: 월 150~300만 원대

**포함 내용**:
- 기장·외부감사
- 이사회 재무 보고
- 지배구조 관리

**조정 기준**:
- 법인 규모와 거래 복잡도

#### 주의사항
```
⚠️ 위 비용은 참고용 가이드이며, 정확한 견적은 상담 후 제공됩니다.
투자 일정, 법인 상황, 제공 서비스 범위에 따라 달라질 수 있습니다.

목적: 가격 비교가 아니라, '이 서비스가 나에게 맞는지' 빠르게 판단하는 용도입니다.
```

## 🎯 사용자 경험 개선

### Before (v7.3.1)
```
Hero → 문제 인식 → 해결책 → 투자 여정
→ ❓ (서비스·비용 불명확)
→ 대표 소개 → 법인 소개 → 문의 폼
```

**문제점**:
- ❌ 어떤 서비스를 받는지 불명확
- ❌ 비용 범위를 모르니 문의 망설임
- ❌ "문의해봐야 아는" 구조

### After (v7.4.0)
```
Hero → 문제 인식 → 해결책 → 투자 여정
→ ✅ 서비스 요약 (단계별 명확)
→ ✅ 비용 가이드 (범위 제시)
→ 대표 소개 → 법인 소개 → 문의 폼
```

**개선점**:
- ✅ 제공 서비스 명확히 이해
- ✅ 비용 범위로 적합성 판단
- ✅ 확신 후 문의 (문의 질 향상)
