# 💼 v8.0.2 PROBLEM 섹션 VC 관점 재작성 완료

## 📋 작업 개요
- **버전**: v8.0.2
- **목적**: VC 실제 판단 흐름에 맞춘 전문가 톤
- **범위**: PROBLEM 섹션 전면 재작성

---

## ✅ 변경 내용

### **1. 헤드라인 재작성**

#### Before (v8.0.1)
```
많은 스타트업이
재무 구조 때문에 투자에서 탈락합니다
```

**문제점**:
- ❌ "탈락" - 공포 조장형 표현
- ❌ 시험 프레임 (합격/탈락)
- ❌ VC의 실제 판단 방식과 불일치

---

#### After (v8.0.2)
```
VC는 과거 숫자가 아니라
재무 구조로 투자 판단을 합니다
```

**개선 효과**:
- ✅ VC 실제 관점 반영
- ✅ 교육적/설명적 톤
- ✅ "판단" - 전문가 언어 사용

---

### **2. 서브 메시지 재작성**

#### Before (v8.0.1)
```
기술이나 비즈니스 모델이 문제가 아니었습니다.
숫자 설명이 부족하고, 재무 구조가 투자 구조와 맞지 않았기 때문입니다.
```

**문제점**:
- ❌ 과거형 ("~였습니다")
- ❌ 문제 나열식
- ❌ VC 관점 부재

---

#### After (v8.0.2)
```
VC는 매출의 크기보다 그 숫자가 어떤 구조에서 나왔는지를 봅니다.
재무 구조가 설명되지 않으면 투자는 보류되거나 더 보수적으로 판단됩니다.
```

**개선 효과**:
- ✅ VC 실제 판단 흐름 설명
- ✅ "보류", "보수적 판단" - VC 실제 언어
- ✅ 구조 중심 설명

---

### **3. 카드 3개 수정**

#### 카드 1: IR 숫자

**Before**:
```
IR 숫자가 스토리를 뒷받침하지 못함
```

**After**:
```
IR 숫자가 투자 스토리를 설명하지 못함
```

**변경 이유**:
- "뒷받침" → "**설명**" (더 명확)
- "투자 스토리" 명시

---

#### 카드 2: 세무 구조

```
세무 구조가 투자 구조와 충돌
```

**유지** (이미 적절한 표현)

---

#### 카드 3: 실사(FDD)

**Before**:
```
실사(FDD) 대응이 안 되어 기회 상실

투자자가 요청하는 재무 자료를 제때 제출하지 못하거나,
실사 과정에서 리스크가 발견되어 밸류에이션이 깎입니다.
```

**문제점**:
- ❌ "기회 상실" - 결과 중심, 공포 조장
- ❌ "밸류에이션이 깎입니다" - 확정적 표현

---

**After**:
```
실사(FDD) 과정에서 추가 질문과 리스크가 반복 발생

투자자가 요청하는 재무 자료를 제때 제출하지 못하거나,
실사 과정에서 설명되지 않는 구조가 발견되어 투자 판단이 지연됩니다.
```

**개선 효과**:
- ✅ "추가 질문과 리스크 반복 발생" - 과정 중심
- ✅ "투자 판단이 지연" - VC 실제 반응
- ✅ 구조 설명 중심
