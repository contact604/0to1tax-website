## 🎯 카카오톡 공유 테스트 방법

### 1. 카카오톡 캐시 초기화
카카오톡은 URL을 캐싱하므로, 배포 후 다음 방법으로 테스트:

#### 방법 1: 개발자 도구
1. https://developers.kakao.com/tool/debugger/sharing 접속
2. URL 입력: `https://0to1tax.com/2026-tax-reform-slides.html`
3. "미리보기 갱신" 클릭
4. 미리보기 확인

#### 방법 2: 실제 공유
1. 카카오톡에서 링크 공유
2. 첫 공유 시 미리보기 생성
3. 이미지/제목/설명 확인

### 2. 네이버 밴드/라인 테스트
- 네이버 밴드, 라인도 Open Graph 지원
- 동일한 방식으로 미리보기 표시
