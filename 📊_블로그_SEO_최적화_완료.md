## 📊 성과 측정 방법

### 구글 서치 콘솔에서 확인
1. **실적 → 페이지**
   - 이 포스트 URL 클릭수 확인
   - 노출수 추이 모니터링

2. **실적 → 검색어**
   - 어떤 키워드로 유입되는지 확인
   - CTR(클릭률) 확인

### 구글 애널리틱스에서 확인
1. **행동 → 사이트 콘텐츠 → 모든 페이지**
   - 페이지뷰 수
   - 평균 체류 시간
   - 이탈률

2. **획득 → 검색 콘솔**
   - 검색어별 유입 분석
   - 랜딩 페이지 성과
