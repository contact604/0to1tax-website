```

│          현 행         │    개 정 안          │

```

#### After (간결한 4열 표)
```
┌─────────────┬────────┬────────┬───────────┐
│    시장     │  현행  │  개정  │   변화    │
├─────────────┼────────┼────────┼───────────┤
│ 코스피      │ 0%     │ 0.05%  │ ↑ 0.05%p  │
```

#### 표 특징
- **4열 구조**: 시장 | 현행 | 개정 | 변화
- **색상 강조**: 개정 열에 파란색 구분선, 개정 세율은 파란 하이라이트 박스
- **변화 표시**: 빨간색 화살표 (↑ 0.05%p)
- **간결함**: 한눈에 파악 가능
