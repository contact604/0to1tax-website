## ✅ 변경사항

### Before
```css
label {
    display: block;
    font-weight: 600;
    color: var(--dark-gray);
    margin-bottom: 10px;
    font-size: 15px;
    /* text-align 없음 (브라우저 기본값) */
}
```

### After
```css
label {
    display: block;
    font-weight: 600;
    color: var(--dark-gray);
    margin-bottom: 10px;
    font-size: 15px;
    text-align: left;  /* ✅ 명시적 왼쪽 정렬 */
}
```
