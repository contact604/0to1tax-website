```bash
git add _posts/2024-12-28-2026-tax-reform.md
```

```
1. FTP 클라이언트 접속
2. 프로젝트 루트 경로로 이동
3. 파일 3개 덮어쓰기:
   - index.html
   - 0to1log.html
   - 2026-tax-reform-slides.html
4. _posts 폴더로 이동
```
