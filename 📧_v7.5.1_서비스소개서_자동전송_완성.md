## 🔧 Netlify Forms 자동 전송

### 폼 이름
```
service-guide-request
```

### 수집 데이터
```
- name: 이름 (필수)
- email: 이메일 (필수)
- company: 회사명 (필수)
- investment_stage: 투자 단계 (선택)
```

### 자동 알림 설정 (2가지 방법)

#### 방법 A: Netlify 기본 알림 (무료)

**설정**:
```
Netlify Dashboard
→ Forms
→ Form notifications
→ Add notification
→ Email: lbg@0to1tax.com
```

**알림 내용**:
```
제목: [서비스 소개서 요청] {이름} - {회사명}
내용: 신청자 정보 포함
```

#### 방법 B: Zapier 자동화 (권장) ⭐

**Zap 구조**:
```
Trigger: Netlify Forms (service-guide-request)
    ↓
Action 1: Gmail - 신청자에게 PDF 이메일 전송
    ↓
Action 2: Gmail - 관리자에게 알림 전송
    ↓
Action 3: Google Sheets - 데이터 기록 (선택)
```

**신청자 이메일 템플릿**:
```
To: {email}
Subject: [Zero to One TAX] 서비스 소개서를 보내드립니다

{name}님, 안녕하세요.
Zero to One TAX 서비스 소개서를 요청해 주셔서 감사합니다.

📎 첨부 파일을 확인해 주세요.

추가 문의: lbg@0to1tax.com | 010-4395-1996

---
Zero to One TAX 선영 회계법인
```

1. **기존 services.html을 PDF로 변환**:

   → 파일명: Zero_to_One_TAX_서비스소개서.pdf

   - Zapier Gmail Action

   - 자동 첨부 설정
