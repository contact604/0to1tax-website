### 4. 핵심 변경사항 (5개 카드)

#### 카드 1: VC 투자의무 기간 확대
```
⏱️ VC 투자의무 기간 확대
투자조합의 자금 집행 유예기간이 확대되어, 보다 여유롭게 투자처를 발굴하고 검토할 수 있습니다.

Before: 3년 → After: 5년
```

#### 카드 2: 펀드별 투자의무 완화
```
📊 펀드별 투자의무 완화
개별 펀드가 아닌 전체 펀드를 기준으로 투자의무를 판단하여, 운용 유연성이 향상됩니다.

Before: 개별 펀드 기준 → After: 전체 펀드 합산 기준
```

#### 카드 3: 민간 벤처모펀드 세액공제 확대
```
🎁 민간 벤처모펀드 세액공제 확대
민간 벤처모태펀드(Fund of Funds) 출자자에 대한 세액공제 혜택이 확대되어, 민간 자금 유입이 촉진됩니다.

개인·법인 모두 소득세·법인세 공제 확대
```

#### 카드 4: SPC 투자도 동일 세제 혜택
```
🏢 SPC 투자도 동일 세제 혜택
SPC(특수목적회사)를 통한 간접투자도 직접투자와 동일한 세제 혜택을 받을 수 있게 되었습니다.

직접투자 = SPC 투자 (세제 동일 적용)
```

#### 카드 5: 모태펀드 존속기간 연장 근거 마련
```
📅 모태펀드 존속기간 연장 근거 마련
모태펀드의 안정적인 운용을 위해 존속기간 연장이 가능한 법적 근거가 마련되었습니다.

필요 시 존속기간 연장 가능
```
