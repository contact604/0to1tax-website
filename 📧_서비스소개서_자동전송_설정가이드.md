## 🔧 Netlify Forms 설정 방법

### Step 1: 배포 후 자동 생성 확인

**배포 후 확인**:
1. Netlify 대시보드 접속
2. 사이트 선택 → **Forms** 탭 클릭
3. `service-guide-request` 폼 자동 생성 확인

---

### Step 2: 이메일 알림 설정

#### 방법 A: Netlify 기본 알림 (무료)

**설정 경로**:
```
Netlify Dashboard
→ Site Settings
→ Forms
→ Form notifications
→ Add notification
```

**설정 내용**:
- **Form**: service-guide-request
- **Event**: New submission
- **Email to notify**: lbg@0to1tax.com

**알림 내용**:
```
제목: [서비스 소개서 요청] {이름} - {회사명}

내용:
- 이름: {name}
- 이메일: {email}
- 회사명: {company}
- 투자 단계: {investment_stage}
- 요청 시간: {submission_time}
```

---

#### 방법 B: Zapier 자동화 (권장) ⭐

**장점**:
- ✅ 신청자에게 자동으로 PDF 이메일 전송
- ✅ 관리자에게도 알림 전송
- ✅ 구글 시트 자동 기록

**설정 순서**:

1. **Zapier 계정 생성** (무료 플랜 가능)
   - https://zapier.com

2. **Zap 생성**:
   ```
   Trigger: Netlify Forms (service-guide-request)
   ↓
   Action 1: Gmail - Send Email (신청자에게)
   ↓
   Action 2: Gmail - Send Email (관리자에게)
   ↓
   Action 3: Google Sheets - Create Row (선택)
   ```

3. **신청자 이메일 템플릿**:
   ```
   To: {email}
   Subject: [Zero to One TAX] 서비스 소개서를 보내드립니다

   {name}님, 안녕하세요.
   Zero to One TAX 서비스 소개서를 요청해 주셔서 감사합니다.

   📎 첨부 파일을 확인해 주세요.
   (PDF 파일 첨부)

   추가 문의사항이 있으시면
   이메일(lbg@0to1tax.com) 또는 전화(010-4395-1996)로 연락 주세요.

   감사합니다.
   
   ---
   Zero to One TAX 선영 회계법인
   서울시 강남구 테헤란로70길 12 H타워 2층
   ```

4. **관리자 알림 템플릿**:
   ```
   To: lbg@0to1tax.com
   Subject: [서비스 소개서 요청] {name} - {company}

   새로운 서비스 소개서 요청이 접수되었습니다.

   - 이름: {name}
   - 이메일: {email}
   - 회사명: {company}
   - 투자 단계: {investment_stage}
   - 요청 시간: {submission_time}
   ```

### Step 3: PDF 파일 준비

**필요한 작업**:
1. `services.html` 페이지를 PDF로 변환
2. Zapier에 PDF 파일 업로드
3. 이메일 전송 시 자동 첨부

**PDF 생성 방법**:
- Chrome: services.html 열기 → Ctrl+P → PDF로 저장
- 파일명: `Zero_to_One_TAX_서비스소개서.pdf`

## 📋 Netlify Forms 데이터 저장

**데이터 필드**:
```
- name (이름)
- email (이메일)
- company (회사명)
- investment_stage (투자 단계)
- submission_time (제출 시간)
```
