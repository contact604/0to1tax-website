### 3️⃣ 반응형 디자인 강화 ✅

#### 📱 모바일 (768px 이하)

**Hero 섹션**:
- 제목: 36px → **32px**
- CTA 버튼: **100% 너비** (터치하기 쉽게)
- 통계: 세로 배치

**대표 소개**:
- 3문장 **가독성 강화** (15px, line-height 1.8)
- 좌우 패딩 감소 (25px → **20px**)
- 텍스트 왼쪽 정렬 (읽기 편함)

**법인 소개**:
- 패딩 최적화 (40px → **35px**)
- 텍스트 크기 조정 (17px → **15px**)

**문의 폼**:
- CTA 버튼: **100% 너비**
- 입력 필드: **16px** (iOS 자동 줌 방지)
- 터치 영역: **최소 44px** (Apple 권장)

**FAQ**:
- 질문 크기: 18px → **15px**
- 패딩 최적화
- 터치 영역 확보
