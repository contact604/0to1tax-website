## 🎯 클릭 플로우 비교

### BEFORE: 2단계 (정보 부족)
```
Step 1: [상담 신청] 클릭
         ↓
Step 2: 상담 폼 작성
         ❌ 정보 없이 결정
```

### AFTER: 3단계 (정보 충분)
```
Step 1: [상담 신청] 클릭
         ↓
Step 2: 모달에서 상세 정보 확인 ← NEW!
         • 가격 확인
         • 전체 서비스 확인
         • 플랜 적합성 판단
         ↓
Step 3: [상담 신청] 클릭
         ✅ 확신을 가지고 결정
```

## 🔄 개발 코드 비교

### BEFORE: 직접 상담 모달 오픈
```html
<button onclick="openConsultModal()">
    상담 신청
</button>
```

### AFTER: 플랜 모달 → 상담 모달
```html
<!-- 플랜 카드 버튼 -->
<button onclick="openStarterPlanModal()">
    상담 신청
</button>

<!-- 플랜 모달 -->
<div id="starterPlanModal" class="modal">
    <div class="modal-content">
        <h3>🌱 스타터 플랜</h3>
        <p>13만원 ~/월</p>
        <ul>
            <li>✓ 무료법인설립자문</li>
            <!-- 8개 서비스 -->
        </ul>
        <button onclick="closeStarterPlanModal(); openConsultModal();">
            상담 신청
        </button>
    </div>
</div>

<!-- JavaScript -->
<script>
function openStarterPlanModal() {
    modal.classList.add('active');
    document.body.style.overflow = 'hidden';
}
</script>
```
