```

**서브 메시지 변경** (분식 오해 방지):
```
✅ "정확한 기장과 세무 신고는 전제입니다.
    문제는 그 숫자를 투자자·감사인이 보는 관점으로 해석하고 설명할 준비가 되어 있는지입니다.
    
    평소에 재무가 관리되지 않으면,
    투자·실사·감사 요청이 올 때마다 반복된 질문과 리스크가 발생합니다."
```

**카드 3개 재작성**:
1. ✅ **숫자는 정확하지만 해석이 준비되지 않음**  
   → 기장은 정확하지만, IR 스토리와 연결 안 됨, 업종 비교 관점 없음
2. ✅ **세무·법인 구조가 평소에 점검되지 않음**  
   → 평소 점검 안 되어 투자 논의 단계에서 리스크 발견
3. ✅ **투자·실사·감사 때마다 급하게 준비**  
   → 평소 투자 관점 정리 안 되어 요청 시마다 자료 급하게 찾는 상황 반복

### 3️⃣ 신규 섹션: "ZERO TO ONE TAX는 이렇게 일합니다" ✅

**목적**: 실제 업무 방식을 명확히 보여주어 신뢰도 향상

#### 4단계 업무 흐름

**1단계: 기장부터 정리**
- 모든 업무는 정확한 기장과 세무 신고를 전제로 시작
- 💡 단순 신고용 장부가 아니라 투자자·감사인이 질문하는 기준으로 숫자 정리

**2단계: 투자·감사 관점에서 숫자 점검**
- 기장된 숫자를 VC·감사인 기준으로 점검
- IR 스토리 연결, 동종 업종 비교, 구조적 리스크 확인
- 💡 법인 구조, 주주 구성, 스톡옵션 설계가 투자 계약과 맞는지 사전 점검

**3단계: 현금흐름과 세무 리스크 관리**
- 현금흐름이 안정적으로 관리되고 있는지 점검
- 합법적인 절세 혜택 활용 + 세무 리스크 사전 관리
- 💡 구조 변경 전 사전 검토: 법인 분할, 주주 변경, 스톡옵션 부여 전 리스크 점검

**4단계: 투자·실사·감사 요청 시 추가 대응**
- IR, 실사(FDD), 감사 요청 시 기존 데이터 기반으로 안정적 대응
- 평소 관리해 둔 데이터 있어 급하게 자료 찾을 필요 없음
- 💡 필요 시 추가 서비스 연계: 기업가치평가, PA, IFRS Conversion

#### Summary Message
```
"평소에 재무를 관리해 두면,
투자·실사·감사 때 흔들리지 않습니다."
```

#### 서브 메시지
```
"반복형 AI 자동 리포트가 아닙니다.
투자 관점으로 숫자를 보는 전문 회계사가 직접 해석하고,
다음 분기에 무엇을 준비해야 하는지 알려드립니다."
```

#### 4가지 특징

**1. 📊 투자자·감사 관점 해석**
- 단순 증감 요약이 아니라 VC·감사인이 질문할 포인트 먼저 정리

**2. 📈 업종별 비교 분석 제공**
- 동종 업종 대비 어느 위치에 있는지, 구조적 차이는 무엇인지 분석

**3. 💰 현금흐름 관리 상태 점검**
- 현금이 잘 돌고 있는지, 구조적 문제는 없는지 회사 건강 상태 점검

**4. 🎯 다음 분기 준비 포인트 정리**
- 다음 분기에 무엇을 준비해야 하는지 구체적 액션 아이템 제시

#### 강조 메시지
```
