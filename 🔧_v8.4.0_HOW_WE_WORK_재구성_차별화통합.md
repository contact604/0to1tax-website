## 🎯 주요 변경사항

### 1. 서브타이틀 2단계 구조

**추가된 서브타이틀**:
```
숫자를 '만드는 것'이 아니라,
대표가 판단할 수 있는 상태로 '관리'합니다.
```

**핵심 메시지**:
- '만드는 것' ❌ → '관리' ✅
- 목표: 대표가 판단할 수 있는 상태

---

### 2. 4단계 업무 방식 - PROBLEM 연계

#### Step 1: 숫자는 정확하게, 하지만 '결정 기준'으로 관리

- **PROBLEM**: 💬 숫자는 있지만 결정 불가
- **해결**: 설명 가능한 구조로 관리, 의사결정 연결
- **핵심**: 기장은 시작일 뿐, 목표는 의사결정

#### Step 2: 분기마다 투자 전문 회계사가 직접 설명

- **PROBLEM**: 💬 설명·커뮤니케이션 부재
- **해결**: 자동 리포트 ❌, 회계사가 직접 해석 및 설명
- **핵심**: 보고서가 아니라, 대표를 위한 분기 점검

#### Step 3: 업종 비교 + 현금흐름으로 회사 상태 점검

- **PROBLEM**: 📊 회사가 잘 가는지 판단 불가
- **해결**: 업종 비교 + 현금흐름 + 세무 리스크 관리
- **핵심**: 잘 가는지, 망할 위험은 없는지 분기마다 점검

#### Step 4: 요청이 와도 '처음부터 다시' 하지 않음

- **PROBLEM**: ⏰ 투자 시점에 바꾸기엔 늦음
- **해결**: 평소 관리된 데이터 기반 대응
- **핵심**: 이미 준비된 상태로 들어감

---

### 3. 차별화 섹션 신규 추가

**제목**: 🔗 커뮤니케이션 & 관리 방식  
**부제**: 기장 대행이 아니라, **상시 재무 파트너 방식**

#### 카드 ①: 고객사 전용 웹사이트 운영 🌐
- 기장 + 보고서 + 대응 내역 통합 관리
- 끊김 없는 히스토리
- 담당자 변경에도 설명 가능

#### 카드 ②: 카카오톡 전용 커뮤니케이션 채널 💬
- 즉시 커뮤니케이션
- 쌓아두지 않고 바로 공유
- 대표의 고민 즉시 반영

```
  └─ ② 투자 시점에 바꾸기엔 너무 늦다
          ↓
HOW WE WORK (4단계)
  ├─ Step 1 → PROBLEM ① 해결
  ├─ Step 2 → PROBLEM ① 해결
  ├─ Step 3 → PROBLEM ① 해결
  └─ Step 4 → PROBLEM ② 해결
          ↓
차별화 요소
  ├─ 전용 웹사이트
  └─ 카카오톡 채널
```
