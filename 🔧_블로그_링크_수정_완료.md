# 🔧 블로그 포스트 링크 수정 완료

## 문제 상황

블로그에서 "2026년 세법 주요 개정사항" 포스트를 클릭하면:
```
blog-detail.html?slug=2024-12-28-2026-tax-reform
    ↓
❌ "게시글을 찾을 수 없습니다" 오류
```

**원인:** `blog-detail.html` 페이지가 존재하지 않거나 마크다운 파일을 읽을 수 없음

## 해결 방법

2026 세법개정 포스트는 **슬라이드로 직접 이동**하도록 특별 처리

### 수정된 로직

```javascript
// 2026 세법개정 포스트 감지
const isSpecialPost = post.slug === '2024-12-28-2026-tax-reform';

// 링크 URL 설정
const linkUrl = isSpecialPost 
    ? '2026-tax-reform-slides.html'      // 슬라이드로 직접
    : `blog-detail.html?slug=${post.slug}`; // 일반 포스트

// 새 탭 열기
const linkTarget = isSpecialPost ? 'target="_blank"' : '';

// 버튼 텍스트
const buttonText = isSpecialPost ? '슬라이드 보기' : '자세히 보기';

// 아이콘
const icon = isSpecialPost ? 'external-link-alt' : 'arrow-right';
```

## 수정된 코드 위치

### 0to1log.html
**라인 748-778** (renderPosts 함수 내부)

```javascript
// 특별 포스트 확인
const isSpecialPost = post.slug === '2024-12-28-2026-tax-reform';
const linkUrl = isSpecialPost ? '2026-tax-reform-slides.html' : `blog-detail.html?slug=${post.slug}`;
const linkTarget = isSpecialPost ? 'target="_blank"' : '';

// 카드 HTML 생성
return `
    <article class="article-card card-cosmic rounded-xl p-6">
        <h3>
            <a href="${linkUrl}" ${linkTarget}>${post.title}</a>
        </h3>
        ...
        <a href="${linkUrl}" ${linkTarget}>
            ${isSpecialPost ? '슬라이드 보기' : '자세히 보기'} 
            <i class="fas fa-${isSpecialPost ? 'external-link-alt' : 'arrow-right'}"></i>
        </a>
    </article>
`;
```

## 다른 포스트와의 차이

### 일반 포스트
- **링크**: `blog-detail.html?slug=...`
- **타겟**: 같은 탭
- **버튼**: "자세히 보기 →"
- **아이콘**: `fa-arrow-right`

### 2026 세법개정 포스트
- **링크**: `2026-tax-reform-slides.html`
- **타겟**: 새 탭 (`target="_blank"`)
- **버튼**: "슬라이드 보기 ↗"
- **아이콘**: `fa-external-link-alt`

## 장점

### 1. 즉시 접근
✅ 별도 포스트 페이지 없이 슬라이드 직접 보기
✅ 클릭 1번으로 전체 내용 확인
✅ 로딩 시간 단축

### 2. 사용자 경험
✅ 새 탭으로 열려 블로그 위치 유지
✅ "슬라이드 보기" 명확한 액션 표현
✅ 외부 링크 아이콘으로 새 탭 열림 예상 가능

### 3. 유지보수
✅ `blog-detail.html` 페이지 불필요
✅ 마크다운 파일은 메타데이터 용도로만 사용
✅ 향후 다른 특별 포스트도 같은 방식 적용 가능

## 추가 특별 포스트 처리 방법

향후 슬라이드나 외부 링크로 연결할 포스트가 있다면:

```javascript
const specialPosts = {
    '2024-12-28-2026-tax-reform': '2026-tax-reform-slides.html',
    '2025-01-15-special-post': 'special-page.html',
    // 추가 특별 포스트...
};

const isSpecialPost = specialPosts.hasOwnProperty(post.slug);
const linkUrl = isSpecialPost 
    ? specialPosts[post.slug] 
    : `blog-detail.html?slug=${post.slug}`;
```
