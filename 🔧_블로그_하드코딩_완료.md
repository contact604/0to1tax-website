# 🔧 블로그 포스트 하드코딩 추가 완료

## 문제 상황

블로그에서 "2026년 세법 주요 개정사항" 포스트가 표시되지 않음

**원인:**
- `_posts/2024-12-28-2026-tax-reform.md` 파일이 서버에 없음
- 마크다운 파일을 로드하려고 시도하지만 404 오류 발생

## 해결 방법

마크다운 파일 없이 **하드코딩된 데이터**로 직접 포스트 추가

### 수정된 코드

```javascript
async function loadBlogPosts() {
    // 2026 세법개정 포스트 (하드코딩) ⭐
    const taxReformPost = {
        slug: '2024-12-28-2026-tax-reform',
        title: '2026년 세법 주요 개정사항',
        date: '2024-12-28',
        author: 'Zero to one TAX',
        category: '세무',
        tags: ['세법개정', '법인세', '고용세액공제', '배당소득', '증권거래세'],
        thumbnail: '',
        excerpt: '고용·출산 인센티브 강화부터 배당 과세 완화까지, 2026년 세법개정안의 모든 것을 17개 슬라이드로 한눈에 확인하세요.',
        body: '',
        published: true
    };
    
    // 다른 포스트들 로드 (마크다운 파일에서)
    const posts = await Promise.all(
        BLOG_POSTS
            .filter(f => f !== '2024-12-28-2026-tax-reform.md')  // 제외
            .map(async (filename) => {
                // 마크다운 파일 로드...
            })
    );
    
    // 세법개정 포스트를 맨 앞에 추가 ⭐
    const allLoadedPosts = [taxReformPost, ...posts.filter(post => post !== null)];
    
    // 날짜순 정렬
    allPosts = allLoadedPosts.sort((a, b) => new Date(b.date) - new Date(a.date));
}
```

## 작동 방식

### 1. 하드코딩된 포스트 생성
```javascript
const taxReformPost = {
    slug: '2024-12-28-2026-tax-reform',
    title: '2026년 세법 주요 개정사항',
    date: '2024-12-28',
    category: '세무',
    tags: ['세법개정', '법인세', ...],
    excerpt: '고용·출산 인센티브 강화부터...',
    published: true
};
```

### 2. 마크다운 파일 목록에서 제외
```javascript
BLOG_POSTS.filter(f => f !== '2024-12-28-2026-tax-reform.md')
```
→ 이 파일은 로드하지 않음 (없어도 괜찮음)

### 3. 포스트 배열에 추가
```javascript
[taxReformPost, ...다른_포스트들]
```
→ 세법개정이 맨 앞에 위치

### 4. 날짜순 정렬
```javascript
.sort((a, b) => new Date(b.date) - new Date(a.date))
```
→ 2024-12-28이 가장 최신이므로 최상단 표시
