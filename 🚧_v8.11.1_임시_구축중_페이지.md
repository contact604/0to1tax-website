## 🔄 복구 방법 (완성 후)

### 방법 1: 파일 이름 변경 (간단)
```bash
# 1. 임시 페이지 보관
mv index.html index-temp.html

# 2. 작업 파일을 메인으로
mv index-working.html index.html

# 3. GitHub에 푸시
git add index.html
git commit -m "Deploy: Full website live (v8.11.0+)"
git push origin main
```

### 방법 2: 파일 복사 (원본 유지)
```bash
# 1. 작업 파일을 메인으로 복사
cp index-working.html index.html

# 2. GitHub에 푸시
git add index.html
git commit -m "Deploy: Full website live (v8.11.0+)"
git push origin main
```

### 방법 3: 직접 교체 (가장 확실)
1. 로컬에서 `index-working.html` 내용 복사
2. `index.html`에 붙여넣기
3. 저장 후 GitHub에 푸시
